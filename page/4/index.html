Windows 7/8/10/11 系統 - 免啟動系統去除桌面中的 "啟用 Windows 水印"
發表於2022-06-12|更新於2022-06-13|Windows|撳制
前言Blog 主我絕不會推薦流氓軟件達至修改電腦，本教學不需使用任何第三方軟件。 開始當你仲未啟用 Windows 既時候，你右下角會出現 啟用 Windows移至 [設定] 以啟用 Windows。 要移除我們需要修改 regedit (登錄檔)。以下教學。 開啟 Regedit 登錄檔呢次我地同上一個 Post 唔一樣，上次係可以分開咁 set，今次就要用 系統管理員 開啟你既 Regedit。 進入路徑我地入到黎就入呢個 Paths 1電腦\HKEY_CURRENT_USER\Control Panel\Desktop 修改檔案搵一個名叫 PaintDesktopVersion 既檔，撳入去再將原本既數值 0 改做 1，改完之後再確定。 重開電腦呢次既修改係會將你所有 user 都會改變，但係我地呢次係需要重新開你部電腦，當你再開機既時候就會發現，你原本右下角既水印係已經消失左。 如果你想開返既話只需要將 PaintDesktopVersion 改返做 1 就得了。

發表於2022-06-12|更新於2022-06-13|Windows|撳制
前言Windows 10 入面，如果你未啟動係用唔到一啲功能，例如係轉唔到你既主題 (Themes)，佢會話 “你必須先啟用 Windows，才能個人化電腦。” 開啟 Regedit首先我地喺 Windows 查 regedit (登錄檔)， 注意 : 用系統管理員行程式係唔同用普通用戶行程式，兩個係分開，如果你改左喺系統管理員既野，咁只會喺管理員個 Account 度生效，並唔會喺普通用戶度生效。 進入路徑黎到呢度我地入呢個 Paths， 1電腦\HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize 修改檔案我地搵一舊叫做 AppsUseLightTheme，將佢打開再將入面既數值改做 0。 成功當你改完撳確定，你會發現已經幫你轉左做 Dark Themes。
