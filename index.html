Windows 7/8/10/11 系統 - 免啟動系統去除桌面中的 "啟用 Windows 水印"
發表於2022-06-12|更新於2022-06-13|Windows|撳制
前言Blog 主我絕不會推薦流氓軟件達至修改電腦，本教學不需使用任何第三方軟件。 開始當你仲未啟用 Windows 既時候，你右下角會出現 啟用 Windows移至 [設定] 以啟用 Windows。 要移除我們需要修改 regedit (登錄檔)。以下教學。 開啟 Regedit 登錄檔呢次我地同上一個 Post 唔一樣，上次係可以分開咁 set，今次就要用 系統管理員 開啟你既 Regedit。 進入路徑我地入到黎就入呢個 Paths 1電腦\HKEY_CURRENT_USER\Control Panel\Desktop 修改檔案搵一個名叫 PaintDesktopVersion 既檔，撳入去再將原本既數值 0 改做 1，改完之後再確定。 重開電腦呢次既修改係會將你所有 user 都會改變，但係我地呢次係需要重新開你部電腦，當你再開機既時候就會發現，你原本右下角既水印係已經消失左。 如果你想開返既話只需要將 PaintDesktopVersion 改返做 1 就得了。

Cyber Security - 安裝作業系統後必須要做的安全事項
發表於2022-06-11|更新於2022-06-11|三修系統大撚哂|Cyber Security
前言當你剛剛安裝好你的作業系統後，以安全角度是並未做好設置，你必須進行以下操作來為你的系統有更進一步的安全。 第一樣 本機帳號加入 一般用戶 權限的帳號 安裝後，部份系統是沒有為你建立普通權限的本機帳號，像是 Windows 這樣的作業系統，預設是使用 系統官理員 的帳號，他並沒有為你建立普通帳號，當你什麼事情也是使用 系統官理員 身份進行，在電腦世界是相當危險，所以你要做的是，為你的系統建立一個 一般權限 的帳號，平時使用 一般權限 的帳號，需要使用 官理員 權限才切換帳號，這樣才是最好的做法。 第二樣 瀏覽器安裝插件 你平時上網的時候，每個網站也有各種不同的插件和腳本，你並不知道當中加入了什麼，有可能是有害你電腦的插件，有毒的廣告，木馬等等。 建議安裝以下插件。 Privay Badger 將追蹤網站封鎖。 ublock Origin 封鎖不同的有害連結。 HTTPS Everywhere 將你瀏覽的網站加入 HTTPS，將你和網站的連結加多一層保護 (加密連線)，防止 Mitm Attack (中間人功擊) 等。 以上就是我個人認為必須要做的事情。
