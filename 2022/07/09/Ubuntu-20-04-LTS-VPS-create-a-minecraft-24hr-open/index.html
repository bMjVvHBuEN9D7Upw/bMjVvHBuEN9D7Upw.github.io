Ubuntu VPS Server - 建立 Minecraft 伺服器並廿四小時開放不停止

發表於2022-07-09|更新於2022-07-12|Minecraft

開啟伺服器需求

如要開啟伺服器，必須開放目前主機的 IP 位置並且開放一個 Port 位置讓其他人可以進入到你的伺服器。
本篇文章會使用以下的工具開啟 Minecraft 伺服器 ~

  • VPS Server - 2RAM 的配置 (最小要用 2RAM)
  • SSH - 連接伺服器工具 (必要)
  • Filezilla - Gui 工具放置檔案用 (可選)
  • SCP - 指令放置檔案用 (可選)
  • Java Edition server - Mincecraft 必須要用 (必要)
  • Screen - 後台運行伺服器程式 (必要)

如果不懂如果建立 VPS Server，可以參考我這篇使用 Digitalocean 建立的文章：Digitalocean - 建立 Droplets VPS Server

第五步 - 安裝 Java JDK

接下來，安裝 Java 的 OpenJDK 版本 17 (JRE)，這是 Java 的最小版本，它刪除了對 GUI 應用程序的支持，所以在 VPS 上運行是非常適合，去除沒必要的檔案大小 ~

另外，OpenJDK 版本 17 是支援 Minecraft 最新版本的 1.19 ~~

第六步 - 安裝 Screen

這工具會使你可以在後台運行伺服器，應該是叫做分離視窗，即使你將視窗關閉也不影響你伺服器的運行。

第十步 - 啟動伺服器

輸入以下指令：

  • Xms1024M - 伺服器配置會以 1024MB 或 1GB 的 RAM 開始運行，如果您希望伺服器以更多 RAM 啟動，可以提高數字。
    M 代表兆字節，G 代表千兆字節都是受支持的選項。

  • Xmx1024M - 這會將伺服器配置為最多使用 1024M 的 RAM，如果您希望伺服器以更大的 RAM 運行，允許更多玩家，或者如果您覺得服務器運行緩慢，您可以提高此限制。

  • jar - 此標誌指定要運行的服務器 jar 文件。

  • nogui - 這告訴伺服器不要啟動 GUI，因為這是一個 VPS 伺服器，而且你沒有圖形用戶界面 (gui)。

第十二步 - 修改 eula.txt 檔案

現在我們需要修改以下兩個檔案：

  • eula.txt - EULA (End User License Agreement)

nano 教學：Ctrl+S 儲存，然後 Ctrl+X 是退出。

第十三步 - 修改 server.properties 檔案

現在，我們看看 server.properties 檔案的內容，這個檔案會包含你伺服器的一些 config，例如 伺服器名稱, 伺服器 IP 位置, PVP 模式開關, 模式。

尋找以下三樣東西，像以下這樣子，確定已經更改為你想要的內容後，儲存檔案後退出。

  • rcon.password=hi

  • enable-rcon=true
