Windows 10 - 完全關閉內建防毒程式

發表於2022-07-02|更新於2022-07-02|Windows

前言 (一)

對於煩人的防毒軟件，主要原因不外乎四個原因。

  • 令你電腦變慢，多餘的工具。
  • 檔案誤判為有毒，非常麻煩。
  • 私隱問題，檔案被掃瞄沒有私隱。
  • 不想見到它 XD

本文章會教你如何將內建的防毒軟件完全關掉。

前言 (二)

雖然關閉防毒對於電腦的效能一定會有明顯的上升，但對於不懂電腦的人來說，我這邊是不建議將防毒去除，我會建議安裝其他第三方的防毒軟件，例如：

  • Malwarebytes：一個殺毒軟件。
  • Bitdefender：一個防毒軟件。

未關閉前

可見有非常多功能，防毒會隨便將你的檔案進行一次掃瞄，隨著時間關係，硬件的衰佬會因為防毒的關係急速下降，令到電腦變得很卡，對於一個懂電腦的人來說，防毒可以說是可有可無，意識遠遠大於工具。

另外，如果只是使用官方的教學將防毒關閉，也會不知道什麼時候突然自動開啟。

gpedit.msc

以下操作需要使用 系統管理員身份 進行。

建議：有關 系統管理員 的內部操作，使用 系統管理員帳號 更變會更好。一般帳號只用於一般操作，兩者分開會最好。

尋找 gpedit.msc

直接於右下角輸入 gpedit.msc 尋找，並使用 系統管理員身份 執行。

電腦設定

雙擊 Mouse 點擊 電腦設定。

系統管理範本

雙擊 Mouse 點擊 系統管理範本。

Windows 元件

雙擊 Mouse 點擊 Windows 元件。

Microsoft Defender 防毒軟體

雙擊 Mouse 點擊 Microsoft Defender 防毒軟體。

關閉 Microsoft Defender 防毒軟體 (一)

雙擊 Mouse 點擊 關閉 Microsoft Defender 防毒軟體。

關閉 Microsoft Defender 防毒軟體 (二)

將原本的 尚未設定(C)，改為 已啟用(E)，然後點擊 套用(A)，再點擊 確定。

設定

接下來我們需要將原本已開啟的防毒相關功能都關掉。

說明：以下操作一樣會需要使用 系統管理員。

開啟 設定

直接在左下角輸入 設定 尋找 設定 然後將他開啟。

更新與安全性

點擊 更新與安全性。

Windows 安全性

點擊 Windows 安全性。

病毒與威脅防護

點擊 病毒與威脅防護。

病毒與威脅防護 (管理設定)

點擊 管理設定。

病毒與威脅防護設定

將所有的已開啟項目全部關閉

注意：需要使用 系統管理員。

病毒與威脅防護設定 (未關閉前)

下圖為預設的 Windows 防毒設定，我們需要將所有已 開啟 的都通通關掉。

病毒與威脅防護設定 (己關閉後)

下圖為關閉後的畫面。

重新啟動電腦

完成以上的步驟後，將電腦重新啟動，然後就成功了。

注意：不是關機再啟動電腦，是重新啟動。

查看是否關閉成功

當成功的時候，開啟 病毒與威脅防護 後會顯示 正在取得保護資訊...，

然後就會顯示 你的病毒與威脅防護是由您的組織所管理。
