Digitalocean - 儲存現時狀態和還原狀態

發表於2022-07-12|更新於2022-07-12|Digitalocean infoDroplet

咩係 Snapshots？

首先喺用之前，你要明白咩係 Snapshots，我會用一個情況黎解釋呢個 terms，喺現實世界，好多野你做左冇得返轉頭，你冇得打回原型，例如食左既野冇得再喺個胃拎返出黎，做左啲戇鳩野俾人笑一世，總之一句講哂，就係你冇得返去你未做之前既狀態。

電腦 Snapshots 就係一個可以俾你打回原型，返返去 save point 個個位，即係好似打機咁，你喺呢度 create 一個 point，你想之後返返去讀點？無問題。你想返返去之前個位？無問題。
你只要有 save 到個 point 就可以返去喇.

留意返 用呢個功能其實係收錢㗎 xd

其實如果要用呢個功能既話 DigitalOcean 係會計你錢㗎，佢係 0.06 美金/GiB/月費，然後 Droplet 每GB 既 Data 會最多需要用一分鐘時間去進行處理，仲有就係會根據你而加部機入面有幾多 GB 再計 0.06 美金/Gib/月.
不過我呢篇文都係做個示範講解下點用 Snapshots 姐 haha

第一步 - 入 DigitalOcean Droplets 主頁

呢度我會預你本身已經起好左一部機黎用，至於點樣起部機唔講啦，
如果唔識既話可以睇返我以前呢篇文：Digitalocean - 建立 Droplets VPS Server

返返黎先，我地入呢個位，喺左邊既咩料入面撳 Droplets，然後撳返你要整 Snapshots 個部機.

第二步 - Dashboard 入 Snapshots 個位

黎到呢個位會入左去你部機既 Dashboard，喺下面張圖你會見到我紅色圈起左 Snapshots，撳佢一下之後就去下一步.

第三步 - Take live snapshot

喺 save point 之前，俾大家睇下我部機入面有乜野先，之後會加啲 file 喺部機入面，再試下 restore 佢返去 save point 位.

我喺部機既 shell 打 ls -la 睇下呢個 path 有乜料到先.

我地可以睇到寫住 total 得 32，而加我地將部機熄左先，咁做係確保數據係一致性，打呢句指令：

而加我地去返 snapshot 個位，幫呢個 snapshot 改個名，然後再撳 take live snapshot，撳完之後等佢 load 一陣就可以見到架啦 ~

load 好喇 ~~

第四步 - 試下起啲野

我地而加登入返個 shelll，然後試下起野，我會起一個 folder 叫 testing.

第五步 - Restore Droplets

而加我地返返去 take snapshot 個位，喺頭先起左一個 snapshots 個度，最右邊撳 More，撳完之後你會見到 Restore Droplet.

撳完之後佢會問你係咪 confirm 要 restroe? 你撳 Restore Droplet 繼續.

呢個 moment 只需要等一陣就得 (大概廿秒)，而喺同一時間，你會發現你個 shell 係打唔到野，然後過一陣會顯示 Connection reset，個原因係 digitalocean 個系統收到風要將你部機 restore，咁佢就會幫你熄機進行呢個程序.

最後一步 - 登入系統睇下係咪真係成功左

我地登入再打 ls -la 睇下係咪未 save 左個陣既狀態先.

成功左喇，可以見到我頭先 create 既 folder testing 已經唔見左，證明左已經將個狀態變返去之前個個狀態 :3
教學完畢 :3

剷 Snapshots

話哂都要計錢，咁假設你唔想再 hold 住既話，我地去返 digitalocean snapshots 個位，撳 More 之後撳 Delete 就得㗎啦.

之後佢會問你係咪真係要 delete，因為呢個操作係冇得返轉頭，delete 左就冇，撳 Delete Snapshot 繼續.

已經成功剷左呢個 snapshot 啦
