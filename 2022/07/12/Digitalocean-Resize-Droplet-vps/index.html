Digitalocean - Droplet 更改伺服器 Spec (Resize)

發表於2022-07-12|更新於2022-07-12|Digitalocean infoDroplet

咩係 Resize？

當你起完部 VPS 用左一排，用用下發現唔夠位裝野 wor 屌，又或者你部機裝野裝到 lag，你想加多條 ram 等佢行得順啲，但係又唔想由頭裝過哂啲野。
呢個時候我地可以用 Digitalocean 提供左既 Droplet 功能，佢叫 Resize。好似字面咁解，就係將你而加呢部 Droplet 既 spec 改做其他 plan，假設係由本身既 512RAM 轉做 1G RAM。

第一步 - 入 DigitalOcean Droplets 主頁

呢度我會預你本身已經起好一部 VPS，係後期想改個 Spec，如果唔識點樣起 VPS 既人可以參考返相關既文章：
第一篇係講點樣起部 Droplet (VPS)：Digitalocean - 建立 Droplets VPS Server
第二篇係講點樣用內建既 SSH 登入功能起一部 Drople (VPS)：Digitalocean - 使用 SSH Keys 登入免用密碼加強你的伺服器安全性

我地入呢個位，喺左邊既咩料入面撳 Droplets，然後撳返你要整 Resize 個部機.

第二步 - 熄左你部機先

留意返，喺改 Spec 之前我地需要將你部機熄左先，咁先可以改到，然後熄左要等佢一分鐘，再 F5 refresh 一次個 page.

點解要熄機？我用個情況黎比如，你屋企部實機要加多條 RAM，咁你冇理由部機開緊咁拆個電腦黎裝 RAM 㗎嘛，正常諗一諗都知應該將電源完全斷開之後先進行呢個步驟，如果唔咁樣做，一黎唔安全 (有機會電親自己，嚴重可以電死，上網搵單新聞黎睇是但都搵到幾單係咁樣死 … )，二黎有機會搞到啲資料死左.

熄機係有兩種方法，一種係喺 Dashboard 度熄，一種用 SSH 登入入去 用 command line 熄.

注意：我地平時既熄機係會開返，咁係理所當然係咪先，但係你遠控一部 SERVER 既時候，好多新手岩岩整既時候，有時入唔到係有可能係你未開返部機，你係需要去返 Dashborad 睇下部機係咪開左。另外，poweroff 係熄機，但係你熄左之後係需要開返㗎，如果唔係你入唔到㗎，除非你係 reboot ~

熄機方法一 - Dashboard 控制

去返你既 Dashboard 位置，你會喺右邊見到粒 toggle 係熄機開機既掣，撳佢一下.

佢會問你係咪真係要熄機，你撳 Turn off.

佢會 load 一陣.

部機已熄左.

之後你去返 Dashborad 睇就會見到佢個 toggle Status 變左做 OFF.

第三步 - Dashboard 入 Resize 個位

黎到呢個位會入左去你部機既 Dashboard，喺下面張圖你會見到我紅色圈起左 Resize，撳佢一下之後就去下一步.

第四步 - 揀你想要既 Spec

我地可以睇到佢寫住你而加既 Spec 係：Basic / 512 MB / 1 vCPU

今篇文我會做示範所以我就是但揀個最 Cheap 雞既 spec 啦，就揀 $6/mo 個個 plan.

第五步 - 確定改 Spec

你確定係想改呢個 Spec 既話就撳 Resize

我地可以見到已經成功由原本既 512MB RAM 轉左做 1GB RAM.
