VirtualBox - 安裝 MacOS BigSur

發表於2022-01-16|更新於2022-06-17|VirtualBoxMacOS

前言

如果你想在 VirtualBox 內使用 MacOS 系統，
你需要另外輸入指令以開啟 iso 檔，
接下來會全程教學。

下載檔案

喺開始之前，我地需要下載 MacOS BigSur iso 檔案 :

https://www.mediafire.com/file/9vuo3rcmv0r8ag4/MAC_OS_BIG_SUR.rar/file

VirtualBox 建立新主機

我地開 VirtualBox 開 Create 個新既主機，名就你鐘意起乜就起乜。

選擇 Ram 大小

選擇你想要既 Ram 既大小。
如果你部機係用緊 8GB Ram，就建議用 4 GB (4096MB)。
如果你部機係用緊 16GB Ram，就建議係 5 GB 內 (5120MB)。
如果你部機係用緊 32GB Ram，就建議直接用 16 GB (16384MB)。

GB 計算方法

1GB = 1024 MB
1024 * 4 = 4096MB
所以 4GB Ram = 4096MB

建立虛擬硬碟

由於我地係要用新起既虛擬硬碟，所以請選擇 立即建立虛擬主機(C)。

選擇處擬硬碟類型

類型選擇 VDI (VirtualBox 類型)。

硬碟分配

呢邊選擇 動態分配。

硬碟大小

選擇你想要的硬碟大小，然後就完成建立主機了。

設定主機設置

呢度我地要將部機既 Setting 改小小野，包括係 :

  • 剪貼簿設置
  • 開機順序
  • 視訊記憶體
  • 虛理器
  • 圖形控制器

修改共用剪貼簿設置

對住主機撳右鍵去 Setting > 一般，
將共用剪貼簿同拖放改做雙向。

修改開機順序

再黎入去 系統 > 主機版
將開機順序既 軟碟 光碟 Tick 走佢唔要

修改虛理器大小

再黎入去 系統 > 虛理器
將虛理器既 CPU 改做 4

修改圖形控制器

再黎入去 顯示 > 畫面
將 視訊記憶體改做 128 MB
再將 圖形控制器 改做 VBoxSVGA

指令修改

呢度既目的係為左俾 VirtualBox 讀到你個 iso 檔，原本係 macOS 係做左手腳唔俾人就咁 READ。

開啟指令台 (cd VirtualBox)

開你既任何一個可以打指令既工具，然後 cd 入去 你安裝既 VirtualBox 位置。

修改指令

再打呢6句野，
注意，Your VM Name 係要改返做你起既 VM 名稱。

測試開機

完成左之後開你部機再選擇返岩岩下載左既 iso 檔，咁就成功啦。

成功開啟

見到呢個畫面就代表已經成功開啟 macOS。
