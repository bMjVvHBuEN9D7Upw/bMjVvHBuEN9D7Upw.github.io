VirtualBox - 安裝 MacOS BigSur

發表於2022-01-16|更新於2022-01-31|VirtualBoxMacOS

前言

喺開始之前，我地需要下載呢個檔案

  • MacOS BigSur iso Files

下載左之後，我地開 VirtualBox 開 Create 個新既

名就你鐘意起乜就起乜

選擇你想要既 Ram 既大小

1GB = 1024 MB
1024 * 4 = 4096
所以 4GB Ram = 4096

立即建立虛擬主機

選擇 VDI
選擇返我地岩岩下載完既 vmdk 檔

選擇第一個 動態分配

打你要既 HardDisk 大小

然後入部機既 Setting > 一般 >
將共用剪貼簿同拖放改做雙向

再黎入去 系統 > 主機版
將開機順序既 軟碟 光碟 Tick 走佢唔要

再黎入去 系統 > 虛理器
將虛理器既 CPU 改做 4

再黎入去 顯示 > 畫面
將 視訊記憶體改做 128 MB
再將 圖形控制器 改做 VBoxSVGA

然後開你既 PowerShell CD 入去 你安裝既 VirtualBox 位置

再打呢6句野，
注意，Your VM Name 係要改返做你起既 VM 名稱

完成左之後開你部機再選擇返岩岩下載左既 iso 檔，咁就成功啦。

成功。
