Windows - Process Hacker 監控系統資源的工具

發表於2022-01-09|更新於2022-01-31|Windows

介紹

Process Hacker是一款免費、強大、多用途的工具，可幫助您監控系統資源、調試軟件和檢測惡意軟件。
Github : https://github.com/processhacker/processhacker

Features

  • A detailed overview of system activity with highlighting.
  • Graphs and statistics allow you quickly to track down resource hogs and runaway processes.
  • Can’t edit or delete a file? Discover which processes are using that file.
  • See what programs have active network connections, and close them if necessary.
  • Get real-time information on disk access.
  • View detailed stack traces with kernel-mode, WOW64 and .NET support.
  • Go beyond services.msc: create, edit and control services.
  • Small, portable and no installation required.
  • 100% Free Software (GPL v3)

Settings

如果您從 USB 驅動器運行 Process Hacker，您可能還希望將 Process Hacker 的設置保存在那裡。 為此，請在與 ProcessHacker.exe 相同的目錄中創建一個名為“ProcessHacker.exe.settings.xml”的空白文件。 您可以使用 Windows 資源管理器執行此操作：

確保在“工具”>“文件夾選項”>“查看”中取消選中“隱藏已知文件類型的擴展名”。
右鍵單擊文件夾並選擇新建 > 文本文檔。
將文件重命名為 ProcessHacker.exe.settings.xml（刪除 “.txt”）。

Plugins

可以從 Hacker > Plugins 配置插件。
如果您遇到任何涉及插件的崩潰，請確保它們是最新的。
ExtendedTools 插件提供的磁盤和網絡信息僅在以管理權限運行 Process Hacker 時可用。

KProcessHacker

Process Hacker 使用內核模式驅動程序 KProcessHacker 來協助某些功能，包括：

  • 捕獲內核模式堆棧跟踪
  • 更有效地枚舉進程句柄
  • 檢索文件句柄的名稱
  • 檢索 EtwRegistration 對象的名稱
  • 設置句柄屬性

站長的話

呢個Program 絕對係嬴 Windows 本身個 Task manager 十條街。呀唔係，係100條街…
我自己都用左佢幾年，非常之咁推薦大家使用，我自己係 replace 左本身 windows 個 task manager既。
另外，你唔鐘意既話，係可以開返本身 windows 個 task manager 既。
但係注意，個濕鳩 Windows Defender 係有機會話呢個 Program 係有毒。
