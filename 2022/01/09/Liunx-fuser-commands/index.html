Liunx - fuser 查檔案 Socket

發表於2022-01-09|更新於2022-05-20|Linux

介紹

fuser 是用於查詢正在使用某個檔案、目錄或socket的程式，提供有關擁有該程式的使用者和訪問型別的資訊。
fuser工具顯示了使用指定檔案或檔案系統的每個程式的程式 ID (PID)。

安裝

help

檢查指定 Port Number

終止指定 Port Number 執行 tcp的程式

檢視某個目錄的程式

fuser序可以與 -v 選項一起使用，該選項以詳細模式執行該工具。
verbose選項用於在計算機螢幕上生成詳細輸出，因此使用者可以實時檢視實用程式正在執行的操作。

檢視已使用 tcp 或 udp Port的程式
