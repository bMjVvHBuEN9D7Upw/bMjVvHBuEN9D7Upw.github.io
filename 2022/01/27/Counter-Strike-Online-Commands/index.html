Counter Strike Online - 主控台指令

發表於2022-01-27|更新於2022-05-20|Counter Strike Online

FPS On / Off

左上角顯示 FPS 開關
打 0 或 1 開關
1 係開
0 係熄

隻 Game 既場既光度

預設 1 最大 3.5

背景音效音樂音量

隊友頭部顯示名稱

打 0 或 1 開關
1 係開
0 係熄

單一人物模組

打 0 或 1 開關
1 係開
0 係熄

天氣效果

0=關閉、1=毛雨、2=下雨、3=大雨

使用舊式購買選單

屍體停留時間

個條屍淨係留 50秒

左右手顯示

1 係左
0 係右

動態準星

準星會隨著你既狀態而縮大縮細

消除下雨聲音

雷達模式

0 係透明
1 係實底色

玩家陰影

1 係開
0 係關

觀察模式準星

0 係熄
1 係開

準星顏色

設定值係 RGB

半透明準心

準心大小

可以 0 ,1 ,2 ,3 或 auto ,small ,medium ,large 設定

FPS 遊戲最大輸出

Default 係 72
如果你要用 99，要去熄左呢個程式既垂直同步再重新開 Games 先得。

煙霧效果

(0、1、2) 煙霧效果，0=最佳逼真、1=普通、2=最差

光亮度

預設值為2.5

Direct3D 模式開關反方向渲染

0 係熄
1 係開

畫面上各個模型連接的部分的連串渲染

0 係熄
1 係開
