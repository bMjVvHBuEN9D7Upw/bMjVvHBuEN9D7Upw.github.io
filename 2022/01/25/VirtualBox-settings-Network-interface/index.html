VirtualBox - 網路介面卡設定

發表於2022-01-25|更新於2022-01-31|VirtualBox

前言

以下不專業解釋 …

喺 VirtualBox 入面，
如果你要將你呢部機既 Network 同你屋企既 Wifi 喺同一個 interface，
你需要喺設定度由 NAT 改做 橋接介面卡，
咁樣你呢個 Network 先可以 Detect 到你呢部 VM 機既存在。

事關你而加用緊 NAT 既情況係同你部實體機係一個分開獨立 網路介面卡，
所以佢就 Detect 唔到你呢部 VM 喺你既 Network 入面。

開始

我地可以喺你既 Windows 開 CMD 打 ipconfig

見到呢個位就可以睇到，
你部 VM 冇修改過既話就係會用呢個介面卡去上網。

而呢個就係我地屋企用緊既 WIFI，
我地需要去修改到入去呢張卡既 Gateway先得。

詳細說明

所有 VirtualBox 既預設 Create 完既 Machine 都會係用呢個。
除非你改做 橋接介面卡，唔係都係會係用緊佢原先安裝好既 虛擬介面卡。
