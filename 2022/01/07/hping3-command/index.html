Hping3 - 指令化 DoS 工具

發表於2022-01-07|更新於2022-06-17|Kali

前言

hping3 是一款網路工具，可自定義TCP/IP封包並顯示回應，用於網路、連接埠、協定、作業系統、服務類型掃描。

下載 hping3

玩這些的工具我也會建議使用 Linux 系統，我以 apt-get package manager 做下載示範，只需輸入以下指令就可以了。

  • -y : 解釋為自動輸入 yes，因為 apt-get 會問你是否下載，這樣打了 -y 就可以節省你的時間。

參數

  • -c：-發送封包數量
  • -d：封包大小
  • -S：發送 SYN 封包
  • -w：TCP window 大小
  • -p：目標端口，可以指定任意埠
  • –flood：快的發送封包
  • –rand-source：隱藏來源位置 (隨機產生)

SYN Flooding

說明 : SYN洪水，一種阻斷服務攻擊

功擊主機 IP

注意 : 請小心使用，被找到可能造成法律問題。

堵塞 CPU 中央處理器

此方法會將流量塞入本機 IP 內的 CPU 中央處理器，令他電腦卡卡的。

堵塞上網速度

此方法會將流量塞入本機 IP 的上網服務 (80 端口)，從而令到他的上網速度更慢。
