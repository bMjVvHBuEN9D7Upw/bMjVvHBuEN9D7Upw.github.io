Linux - 修改 Terminal linux@user Display 顏色

發表於2022-01-28|更新於2022-02-02|Linux

前言

今篇文係講點樣改 Linux 既 Terminal 原本個 show user host 隻色 (Foreground).

開始

首先我地開你既 Terminal，
然後再打

然後我地鍵盤 撳 Ctrl + w
查 33，
搵到呢段野.

呢到呢個位我地用鍵盤既 上下左右箭嘴 控制而加既畫面，
同平時一樣咁用 Backspace del野，
我地將

入面既 32m

改做 31
PS : 31 係代表紅色咁解

另外如果你唔想將佢變成紅色既話
可以打其他數字

  • 31 紅色
  • 32 綠色
  • 33 黃色

睇返詳情可以喺呢個網睇
https://misc.flogisoft.com/bash/tip_colors_and_formatting

我地完成哂之後就要 save

喺 用 nano editor 既情況下，
係同平時 gui 用 txt editor 係一樣既，
我地撳 Ctrl + S Save，
完成左之後再撳 Ctrl + X Exit

搞掂之後開一個新既 Terminal 就會發現改左隻色啦！

PS : Termnial 既 更改唔會係即時睇到改變，
你需要重新開過 Terminal 先會睇到效果，
又或者再行多次個 Program.
