Macchanger - 控制 Mac Address

發表於2022-01-30|更新於2022-06-19|Kali

前言

GNU MAC Changer 是一個使網絡接口的 MAC 地址操作更容易的實用程序，以提供一種跨多個網絡跟踪計算機的簡便方法。
通過定期更改 MAC 地址，可以阻止這種跟踪，或者至少使這種跟踪變得更加困難。

MAC 位置是什麼？

MAC 地址是網絡上的唯一標識符，亦可以在大多數網絡硬件上更改。

Mac Changer

網路名稱說明

eth0 : 呢個名稱係好大部分既通常名稱，喺 Linux 世界入面，eth0 代表 lan 卡.. eth0, eth1 … etc
wlan0 : 呢個名稱係好大部分既通常名稱，喺 Linux 世界入面，wlan0 代表 wifi卡.. wlan0, wlan1 … etc

Mac Addrees 更改

Random 更改 MAC Address

隨機咁改 mac address，-r = random。

  • 綠色圈 既位置係你上一次改完既 Mac Address 既位置。
  • 黃色圈 既位置係你原本既 Mac Address。
  • 暗紅色圈 既位置係你今次改完 Mac Address 既位置。

還原本來 Mac Address

-p = permanent

  • 綠色圈 既位置係你上一次改完既 Mac Address 既位置。
  • 黃色圈 既位置係你原本既 Mac Address。
  • 暗紅色圈 既位置係你今次改完 Mac Address 既位置。

指定 Mac Address

-m = mac，然後輸入六組數雙位數字。
