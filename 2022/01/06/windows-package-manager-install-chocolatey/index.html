Chocolatey - Windows Package Manager

發表於2022-01-06|更新於2022-06-11|Windows

Chocolatey 官網

  • https://chocolatey.org/

開始

Step 1

首先開你既 PowerShell (一定要係PowerShell)，
然後要用 Admin 身份開。

輸入指令 Get-ExecutionPolicy，
如果出現 Restricted，
請輸入 Set-ExecutionPolicy AllSigned 或者 Set-ExecutionPolicy Bypass -Scope Process，
如果不是，直接使用 Step 2 中的指令進行安裝 scoop。

Step 2

裝到之後你喺 Terminal 打 choco 就可以用到.
