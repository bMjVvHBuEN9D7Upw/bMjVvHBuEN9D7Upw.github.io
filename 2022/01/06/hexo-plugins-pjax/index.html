Hexo - 安裝 Pjax (局部網頁更新)

發表於2022-01-06|更新於2022-02-18|NPMHexo

前言

呢篇文會以 Next Theme 做示範，
每個 Theme 做既野都大同小異，
所以唔洗太膽心。

pjax 介紹

簡單黎講就係唔洗成個 Page Refresh 就可以 update 個網頁，改埋個 URL，
網站載入速度更加快，
就好似連登呢啲網都係有類似效果。

安裝

入去 /themes/next/_config.yml，
將 pjax: false 改做 true

然後 cd 入去你個 blog 再打

之後重開 server，
你再試下 F5 Refresh 就會感覺到個網快左好多。

參考資料
https://ed521.github.io/2020/05/hexo-next-upgrade/
