Linux - Apache 伺服器

發表於2022-02-16|更新於2022-02-17|Linux

前言

Linux Apache 係可以用黎 test web，
俾同一個 Network 底下既人入。

更新 Package

安裝 Apache2

Apache Status

安裝完成次後我地睇下 Apache2 既 Status.

當你想開左佢既時候，
你可以咁打

當你想停左佢既時候，
你可以咁打

當你想重開佢既時候，
你可以咁打

開啟 Browser

首先我地要知道你開左 Apache 既呢部機既 Private IP 係乜，
我地打 ifconfig 查下
然後 inet 隔離個 IP 就係你呢部機既 Private IP.
例如你見到既 IP 係 192.168.0.205
我地將呢個 IP 打入去 Browser，
就會開到。

檔案位置

Apache 既檔案係會擺喺 /var/www/html
你只要擺入去就得，
就例如你想快速放檔案俾人，
但係又冇 USB，
又唔想用人地個啲 Online files Share web，
咁你只要係喺同一個 Network 底下就可以做到。

假設情況係想將 files 由 電腦A 送去 B 點，
假設檔案名係叫 on99.exe，
你只要將你呢個檔案 copy 去 /var/www/html/

然後當你用部機上 (電腦A Private IP，假設 IP 係 192.168.0.205)
電腦B 我地用部機開 Browser 打

咁你就成功 Access 到，然後再下載個檔案。
