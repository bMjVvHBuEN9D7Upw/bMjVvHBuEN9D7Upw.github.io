Git - .gitignore 排除上傳某啲檔案

發表於2022-02-02|更新於2022-02-02|Git

前言

.gitignore，簡單黎講係排除檔案用既。

開始

建立檔案

首先起個 files 名叫 .gitignore

然後打你想 排除既檔案，

單一檔案

例如你唔想 push files on99.txt

咁只需要喺入面加一行叫做

on99.txt

跟副檔名

又或者你唔想見到全部 txt files

你可以加一行

*.txt

指定路徑檔案

又或者你想排除 指定路徑既檔案，

就咁樣打。

on99/dllm.exe

指定路徑全部檔案

又或者你想喺 指定路徑入面既全部檔案

就咁樣打。

dllm/*

清除

如果你唔想再排除佢地，
你可以用 clean 清走哂佢地。

強制性咁就係

總結
