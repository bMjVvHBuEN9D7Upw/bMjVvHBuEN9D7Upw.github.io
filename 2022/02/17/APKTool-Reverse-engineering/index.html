前言

APKTool 係一款用黎將 APK 檔 decode，重新整成新既 APK 既程式。
即係可以攞到呢個 APK 既 Source code，再將佢修改 (前提你要識睇 Java,XML)，然後再整成新既 APK。

Download APKTool

呢個方法係唔洗用到 Package Manager，亦都係官方教既下載方法

首先我地要確保你部機係裝左 java jdk，
未裝既可以去 https://www.oracle.com/java/technologies/downloads/
Debian 用家可以 download 佢個 .deb 檔
然後打

裝完之後再 download https://raw.githubusercontent.com/iBotPeaches/Apktool/master/scripts/linux/apktool
將佢 save 做 apktool

再去 https://bitbucket.org/iBotPeaches/apktool/downloads/
下載最新既 jar 檔
download 完之後個名改返做 apktool.jar，

然後我地需要改改呢兩個檔案既檔案權限

再將呢兩個檔案 copy 去 /usr/local/bin 呢個位置

完成次後我地再喺 Terminal 打 apktool
咁就可以用啦。

簡單一行裝完。

一樣我地係需要裝 JDK 先可以行到 APKTool.

裝好 JDK 之後 再下載 https://raw.githubusercontent.com/iBotPeaches/Apktool/master/scripts/windows/apktool.bat
將佢 既名 save 做 apktool.bat

再去 https://bitbucket.org/iBotPeaches/apktool/downloads/ 搵最新既 jar 檔，
下載完再將佢 改名做 apktool.jar

完成之後再將呢兩個檔案 copy 去 C://Windows
之後你 cmd or powershell 打 apktool 就會有反應㗎啦。
