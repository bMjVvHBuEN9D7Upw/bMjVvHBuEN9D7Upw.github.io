Android Studio - .isEmpty() | 檢查 EditText 入面係咪冇打野

發表於2022-02-17|更新於2022-05-20|Android Studio

前言

如果你好似平時咁 if input == ""，
咁會出現一個問題，
就係當用戶打 " "
入面係佢打空白鍵，
咁既時候佢都會當你唔係冇打野，
要解決呢個問題有好多方法
而今篇，會講用 .isEmpty() 判斷 EditText 入面係冇打野，

開始

原理

拎 EditText 入面既料
然後再用 .isEmpty() 呢個 function
當如果呢舊野入面係冇打既野時候，
條件就會成立，
然後就判斷為冇打到野。

而 else 就係相反，
你有打到野。

基本上所有字都計，
除左你撳 空白鍵 唔計。

分別大同小異。
