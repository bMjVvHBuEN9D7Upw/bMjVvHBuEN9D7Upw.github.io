Github - 生產個人 access tokens

發表於2022-02-15|更新於2022-06-13|Github

前言

要 ACCESS Private Github Project，
係需要用到你既個人 tokens，
今篇文就係教你點樣起個 personal tokens，
然後再 access 你既 Private Project.

生產個人 Access Tokens

首先去 Setting，
拉到最底搵 <> Developer Settings，
撳 Personal access tokens，
再撳 Generate new token

Note 就係要打你既 名稱，是但起你認到就得。
Expiration 係呢條 keys 可以用到幾耐，

Select scopes 就 Tick 返

  • repo
  • admin:repo_hook
  • admin:gpg_key

Tick 哂之後撳 Generate token

之後你就會見到你既 tokens keys啦 ！

呢個 tokens keys 就係你用黎登入既密碼，
並唔係用你本身登入 github 個密碼黎 access ！

git clone

而加我會示範點樣 git clone 你既 repo.

首先我地要 confirm 你既 config 係打左你既 username,email
打呢句查下係咪有野

set username 入cfg.

set email 入 cfg.

然後照正常咁樣打，
但係密碼會變左你岩岩整完既 tokens keys.
