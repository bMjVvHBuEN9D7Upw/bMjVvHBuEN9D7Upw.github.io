發表於2022-02-08|更新於2022-05-30|Browser

瀏覽器插件

讓你用瀏覽器更加得心應手。

推薦 Web Browser Extensions For 其他 (排名不分先後)

以下連結會帶你去到官方網站或者 Github。

  • 1. Flagfox
  • 2. Imagus
  • 3. Popup Blocker
  • 4. Search by Image
  • 5. SearchPreview
  • 6. Simple Tab Groups
  • 7. Tab Session Manager
  • 8. Tabby - Window and Tab Manager
  • 9. Tampermonkey
  • 10.Undo Close Tab
  • 11. Bitwarden - Free Password Manager
  • 12. Youtube Audio
  • 13. YouTube NonStop

1. Flagfox

Flagfox 是一個 Firefox 擴展，通過在 URL 後面顯示相應的國家標誌，以圖形方式顯示打開網站的 Web 服務器所在的國家/地區。

2. Imagus

放大縮略圖，並通過鼠標懸停顯示來自鏈接的圖像/視頻。

3. Popup Blocker

默認情況下嚴格阻止來自任何網站的所有彈出請求。 打開一個通知窗口，允許您接受、拒絕、打開後台頁面的彈窗，或將當前頁面重定向到彈窗地址

4. Search by Image

強大的圖片搜索工具，支持各種搜索引擎，如Google、Bing

5. SearchPreview

SearchPreview 通過在 Google™、DuckDuckGo™、Yahoo™ 和 Bing™ 搜索結果中插入站點預覽圖像（縮略圖）和流行度排名來增強您的搜索結果頁面。

6. Simple Tab Groups

創建、修改和快速更改選項Tab Groups

7. Tab Session Manager

保存和恢復窗口和選項卡的狀態。支持自動保存和 Cloud 同步。

8. Tabby - Window & Tab Manager

Tabby 幫助您輕鬆管理大量窗口和選項卡； 它可以幫助您在選項卡和窗口上快速打開、關閉、移動、固定和執行許多其他操作。 使用 Tabby，您甚至可以一鍵保存所有窗口和選項卡以供日後使用！

9. Tampermonkey

Tampermonkey 是世界上最受歡迎的用戶腳本管理器。

10. Undo Close Tab

允許您通過單擊恢復剛剛關閉的選項卡，此外它還可以在方便的上下文菜單中提供最近關閉的選項卡列表。

11. Bitwarden - Free Password Manager

安全、免費、跨平台的密碼管理工具

12. Youtube Audio

僅流式傳輸來自 YouTube 的音頻以節省帶寬和電池壽命

13. Youtube NonStop

厭倦了讓“暫停視頻。繼續觀看？” 確認對話？
此擴展可以自動化它，因此您可以不間斷地收聽自己喜歡的音樂。
