Fluent Terminal - Windows 10 第三方 Terminal

發表於2022-05-23|更新於2022-05-23|Windows

Fluent Terminal

特徵

  • PowerShell、CMD、WSL 或自定義 shell 的終端
  • 內置支持 SSH 和 Mosh 連接
  • 支持標籤和多個窗口
  • 主題和外觀配置
  • 導入/導出主題
  • 導入 iTerm 主題
  • 可編輯的鍵綁定
  • Search Function
  • 配置外殼配置文件以在不同外殼之間快速切換
  • 資源管理器上下文菜單集成 安裝腳本可以在這裡找到
  • VsCode 擴展 這裡
