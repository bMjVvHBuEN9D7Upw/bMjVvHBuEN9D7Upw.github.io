Virustotal - 線上病毒木馬 Scanner

發表於2022-05-23|更新於2022-06-13|網站推薦

介紹

VirusTotal 聚合了許多防病毒產品和在線掃描引擎，稱為 Contributors。
來自這些貢獻者的匯總數據允許用戶檢查用戶自己的防病毒軟件可能遺漏的病毒，或針對任何誤報進行驗證，
最大 650 MB 的文件可以上傳到網站，或通過電子郵件發送（最大 32MB）。
防病毒軟件供應商可以接收被其他掃描標記但被他們自己的引擎傳遞的文件的副本，以幫助改進他們的軟件，進而改進 VirusTotal 自身的能力。
用戶還可以掃描可疑 URL 並蒐索 VirusTotal 數據集。
VirusTotal 使用 Cuckoo 沙箱對惡意軟件進行動態分析。

分析 :

  • URL, IP Addrss, domain
  • Files

官網 : https://www.virustotal.com/gui/
