Counter-Strike 1.6 - SteamCMD 建立伺服器並安裝 AMX X Mod (Windows)

發表於2022-05-31|更新於2022-06-13|Counter-Strike 1.6

SteamCMD

Steam 控制台客戶端或 SteamCMD 是 Steam 客戶端的命令行版本，
它的主要用途是使用命令行界面安裝和更新 Steam 上可用的各種專用服務器，
它適用於使用 SteamPipe 內容系統的遊戲。
所有遊戲都已從已棄用的 HLDSUpdateTool 遷移到 SteamCMD。

下載

首先到這網站下載 SteamCMD。

  • https://developer.valvesoftware.com/wiki/SteamCMD#Downloading_SteamCMD

案下 [1] 下載 steamcmd.zip

或者你可以案 這裡

解壓 SteamCMD

將剛剛下載完的 steamcmd.zip 解壓後你會發現一個檔案名為 steamcmd.exe。

建立一個資料夾，再將檔案放到該資料夾(你不這樣做的話檔案會即時生產到目前目錄。)

SteamCMD 下載需要檔案

開啟 steamcmd.exe，
開完之後只需等待他完成。

SteamCMD 指令操作

這裡需要兩個指令

輸入 login anonymous

輸入 app_update 90 validate

  • 90 代表 Counter Strike 1.6

這裡會出現 Error，
但沒關係，
這是 steamcmd 問題，
我們只需再輸入一次就可以。

測試開啟伺服器

完成以上的程式我們可以將 steamcmd.exe 關掉，
然後再嘗試開啟伺服器，
先到以下的路徑 steamcmd\steamapps\common\Half-Life
開啟 hlds.exe

再案啟動伺服器 …

然後我們開啟 CS 再輸入上面看到的 IP位置加入到伺服器列表

成功！

進入伺服器 …

完成。

但是現在伺服器是什麼也沒有，
現在我們開啟安裝 AMX Mod X，
到現有的伺服器。

下載 AMXX 需要檔案

先到 AMX Mod X 官方網站。

我們需要下載三個檔案

記得有 Windows 版本就下載 Windows 版本。

  • Counter-Strike Addon Windows
  • AMX Mod X Base Winodws
  • Metamod

https://www.amxmodx.org/downloads.php

下載完成後你會發現到三個 zip 檔。

解壓 base 再解壓 cstrike

這裡你需要留意次序。

注意 :
meta 的話 不用分開次序，
*** 然後請先將 base 解壓後再將 cstrike 解壓 ***

記得要用 cstrike replace base 的檔案。

解壓 Metamod (無分次序)

將 addons copy 去 cstrike 內

完成後回到 \steamcmd\steamapps\common\Half-Life\cstrike 路徑，
將剛剛解壓完成的 addons，
整個放到 cstrike 內。

建立 plugins.ini 檔案

進入到 \Steam\steamapps\common\Half-Life\cstrike\addons\metamod，
如果找不到一個檔案叫 plugins.ini
那就新增一個檔案叫 plugins.ini

新增 win32

再將剛剛新增的檔案，
在裡面打一行文字。

要跟 Steam\steamapps\common\Half-Life\cstrike\addons\metamod\dlls 內的 metamod.dll

修改 liblist.gam 檔案

進到 Steam\steamapps\common\Half-Life\cstrike，
找一個檔案叫 liblist.gam

然後進到你的伺服器看看有沒有反應 (AMXX，meta)。
