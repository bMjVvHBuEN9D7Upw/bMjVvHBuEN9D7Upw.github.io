Counter-Strike 1.6 - 於 Steam CS 安裝 AMX X Mod (Windows)

發表於2022-05-31|更新於2022-06-10|Counter-Strike 1.6

前言

要喺 1.6 度玩唔同既 Mod，
我地需要用到 AMXX，
然後今篇文係講點樣裝 AMXX。

今篇文係講喺你既 Steam CS 入面裝。

另外，因為今篇係用 Windows 裝，
Linux 用家需要安裝另一個檔案。

首先

我地需要 3個檔案，
我地去 AMXX 官網下載。

記得有 Windows 版本就裝返 Windows 版本。

  • Counter-Strike Addon Windows
  • AMX Mod X Base Winodws
  • Metamod

https://www.amxmodx.org/downloads.php

下載完哂三個你會發現都係 zip 檔案黎，
呢度要留意返個次序，

注意 :
然後 metamod 就可以唔洗分次序，
base 一定係要解壓先過 cstrike。

Unzip base 再 Unzip cstrike

我地先 unzip base，
然後再 unzip cstrike。

記得要 用 cstrike replace base 既 files。

Unzip Metamod (無分次序)

將 addons copy 去 cstrike 內

完成之後去返 你既 \Steam\steamapps\common\Half-Life\cstrike 路徑，
將你岩岩解壓完既 addons，
成個擺入去 cstrike 入面。

建立 plugins.ini 檔案

我地喺 \Steam\steamapps\common\Half-Life\cstrike\addons\metamod 入面，
如果搵唔到一個檔案叫 plugins.ini
咁就新增一個檔案叫 plugins.ini

新增 win32

我地再將岩岩新增左既檔案，
喺入面打一行叫。

要跟返呢個路徑 Steam\steamapps\common\Half-Life\cstrike\addons\metamod\dlls 入面既 metamod.dll

修改 liblist.gam 檔案

入去 Steam\steamapps\common\Half-Life\cstrike，
搵一個檔案叫 liblist.gam
再將入面第 14 行既原本

改做

完成 (測試)

搞掂哂以上既野就已經完成安裝 AMXX，
你可以開你既 Cs1.6，
然後 New Games 睇下有冇反應。
