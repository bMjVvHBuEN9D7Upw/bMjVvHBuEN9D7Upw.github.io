Telegram - 通訊程式

發表於2022-05-24|更新於2022-05-24|匿名

Telegram

前言

Telegram (簡稱TG)，
係一個非常方便又可以做到匿名既程式，
申請需要用到電話號碼，
當你申請完既時候，
你可以將電話號碼隱藏。

https://telegram.org/

1. Secret Chats

雖然 Telegram 聊天都是客戶端-服務器加密的，
但默認聊天仍存儲在 Telegram 的服務器上。
這可以方便地讓您在多個設備上訪問您的消息，
但它可能會引發隱私問題。

為了最大限度地保護隱私，您可以創建使用端到端加密的秘密聊天。 (Secret Chats)
這些不存儲在 Telegram 的服務器上，因此您只能使用您的特定手機訪問它們。
如果一方刪除一條消息，它會在兩台設備上消失，
如果您願意，您可以為所有媒體指定自毀期。

最神奇的地方是，
當你使用 手機A 開啟 Secret Chat 的時候，
你再用 手機B 開啟 Telegram 是並不會發現到你在 手機A 開啟的 Secret Chat的，
你只能用 手機A 看到對話。

2. 自動刪除帳號

當你的帳號不使用你所設置的限期外的時候，
帳號將會自動刪除。

選擇有:

  • 一個月
  • 三個月
  • 六個月
  • 一年

3. Chat Folders

這是一個非常方便你對話的功能。

當你的對話有很多的時候，
你使用這功能後可以令你的對話分類，
例如將 Channal，Personal，Bot，
分開成三個 Folder。

4. Save Messages

用於儲存你的所有檔案，文件，
上限是 2GB (TG 最大上限)。

5. Themes (主題)

黑夜主題，明亮主題這些不用說了吧，
但是 Telegram 也有其他 Themes，
你也可以自訂自己的顏色，
又可以將背景 Blur。

6. Emoji

在 Telegram 中，
他有自己特有的圖案，
也有放大的圖案，
我們平常使用的 emoji，
在 Telegram 中是有放大版本的，(Large Emoji)
你案下去的時候也會有反應。

7. Telegram Bot

Telegram 也提供官方 API 給大家，
建立一個機械人使用起來更加方便，
只要你會編程，
你也能創建自己的 Bot，
例如 Youtube 音樂下載器。

支援系統

  • Linux
  • Mac
  • Windows
  • IOS
  • Android
