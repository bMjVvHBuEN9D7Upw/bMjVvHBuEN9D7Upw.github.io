Session Messenger - 匿名加密通訊程式

發表於2022-05-24|更新於2022-05-24|匿名

Session Messenger

前言

Session 是免費和開源的，
是由非營利性 Oxen Privacy Tech Foundation 開發的跨平台去中心化加密即時通訊服務，
它基於開源信號協議。
用戶可以發送一對一和群組消息，
其中可以包括文件、語音筆記、圖像和影片。

https://getsession.org/

特徵

  • No Phone Numbers
  • No Data Breaches
  • No Foodprints
  • Open Sources (Github)
  • Censorship Resistant
  • end to end encrypted messages

申請帳號

Session 使用 Session ID 作為標識符，並通過端到端加密保護與其他 Session 用戶的所有通信。

開啟帳號只需要幾秒鐘，即時一生一串 ID 作為登入帳號，
不用任何個人資料就能輕鬆申請。

支援系統

  • Linux
  • MacOS
  • Windows
  • IOS
  • Android

Session 擁有適用於 iOS 和 Android 的官方客戶端應用程序以及適用於 Windows、macOS 和 Linux 的桌面應用程序。
