Hexo Butterfly - 安裝 Google Analytics

發表於2022-06-15|更新於2022-06-23|NPMHexo

咩係 Google Analytics？

Google Analytics（簡稱 GA），
一個可以追蹤網站流量俾網站 Admin
追蹤網站流量同埋睇下人地上呢個網睇緊乜，
然後 Admin 喺後台就可以睇到人地上緊你既網既邊個位，
簡單黎講即係 一個分析器。

申請 Google Analytics

非常簡單，你只需要有一個 Google Account 就可以。
之後入 Google Analytics 網站 :

https://analytics.google.com

開始測量

撳 開始測量。

帳戶設定

輸入一個名稱，你易認就好了。
然後按 下一個。

資源設定

輸入一個使用者名稱，你易認就好了。
然後按 下一步。

提供商家相關資訊

這邊你填好後，Google Analytics 會幫你分析你的網站會到什麼價位。
墳好後按 建立。

服務條款合約

Tick Checkbox 後按下 我接受。

電子郵件通知

這邊如果不想有通知會發送給你就直接按下下面的 全部取消勾選並儲存。

設定平台

完成上面既步驟，你已經起左一個資源，然後我地選就開始選擇平台，我地撳 網站。

設定資料串流

輸入你既網站同埋串流名稱後按 建立串流。

串流詳細資料

成功建立串流之後，你會見到評估 ID (Tracker code)，
將佢 copy 低，然後我地去你既 hexo themes 修改。

修改 pug 檔案

如果你本身有在 yml 檔的 google_analytics 嘗試加入 評估ID，請將他去除，以免造成錯誤。

由於 Butterfly 原本自己的問題，使用原本 yml 檔內提供的 google_analytics 是沒有反應的，

等待

接下來只是等待他啟動，時間會是 48 小時內。

確定網站有 Google Analytics

使用 ublock Origin 確定有沒有連結 :

www.googletagmanager.com

確定評估 ID

鍵盤按下 F12 > Network 後，
按下 F5 將頁面重新整理 > 再查 google。

然後如果看到你的 G-ID，就代表已經成功，如果沒有顯示，那麼你的設置絕對是搞錯了，請查看自己的檔案哪裡做錯了。

管理後台

如果你想查看後台內容，只需於 Google Analytics 頁面 右下角按下 管理

管理資料串流

按下 資料串流。

管理網站

這樣就返回到這一頁了。
