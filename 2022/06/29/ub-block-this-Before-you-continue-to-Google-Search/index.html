uBlock Origin - 封鎖煩人的要求使用 Cookie 彈出視窗。

發表於2022-06-29|更新於2022-06-29|uBlock Origin

前言

當你使用 Google 尋找內容，然後顯示一個視窗說明 (如下圖)。

Before you continue to Google Search

而本文章會教你如何封鎖這個視窗，令你的瀏覽器不再彈出視窗 要求使用 Cookie。

有關封鎖

要開始封鎖先，我們需要於瀏覽器安裝 ublock Origin 插件，網址如下：

【Google Chrome】: 按這裡
【Firefox】: 按這裡

安裝插件 (Firefox)

第一步 (Firefox)

點擊 Add to Firefox。

第二步 (Firefox)

點擊 Add。

安裝完成 (Firefox)

完成以上步驟後，已經安裝完成，可查看瀏覽器右方會多出了一個圖示，他就是 ublock Origin 的插件圖示，點擊他會出現更多內容。

加入新的 Tags

接下來我們需要新增新的規則以封鎖這個視窗的 HTML Tags

第一步 (開筆)

點擊 uBlock Origin 後，再點擊下方的 筆。

第二步 (選擇移除位置)

點擊後你會發現瀏覽器多變成了紅色，這邊要選擇你想除移的位置，我們將 Mouse 移到最邊邊，然後點擊一下以示確定。

第三步 (視窗彈出)

當你點擊後你會發現多了一個視窗於右下角，這個就是你點擊後他偵測到的 Tags。

第四步 (測試 Tags)

來到這一步我們要進行測試，看看是否移除了我們想要的東西。
點擊 Preview 查看。

第五步 (確定 Tags)

OK，點擊後發現不想要的內容不見了，但是有點怪怪的，就是發現顏色還沒有改變至白色 (Background)，
這表示我們要去除的 Tags 並不是這個，一但選擇了這個，
雖然不見了視窗，但由於這個視窗有幾層的關係，我們需要將他全部去除才能正常使用頁面。

第六步 (Cosmetic filter)

我們點擊下方的 Cosmetic filter 所顯示的 Tags (這邊的 Tags 會自動顯示你有可能想要的 Tags)。
我們每一個都點擊一次看看效果如何，再決定哪一個是真正封鎖全部。

嘗試點擊第二個 ##.HTjtHe，發現畫面都白了，能正常顯示內容。

第七步 (新增規則)

決定好是這個 Tags 後，點擊 Create 將他新增至 Filter 名單當中。

查看新增內容

開啟瀏覽器內的插件設定後然後到 My filter 查看，會發現多了一個規則，
這就是你剛剛按下 Create 的規則，如果你不想要只需要將這行移除就可以了。

注意

Tags 會有機會隨住網站的變動而改變了，所以有可能會突然失效，如果失效你只需要再次做一次以上的操作新增多一個 Tags 就可以了。
