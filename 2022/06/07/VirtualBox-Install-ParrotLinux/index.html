VirtualBox - 安裝 Parrot Linux

發表於2022-06-07|更新於2022-06-13|VirtualBoxParrot

前言

Parrot 係一款用黎主要做 Penetration Testing 既系統，
今日講下點樣下載 Parrot Linux 喺你既 VirtualBox 入面，
至於點樣開個新既 VirtualBox 就唔喺度講，
可以睇返之前既文。

下載 iso 檔案

首先去官網下載 iso 檔案先，
兩個版本都冇乜所謂。

  • Home Edition
  • Security Edition

https://www.parrotsec.org/download/

進入

入左去之後係用你既鍵盤 上下左右箭嘴 進行操作，
我地撳 Try / Install

…. Loading

入到黎之後會發現個 Screen 好細。

我地撳 Host + F 開啟 Full Screen。

安裝 (系統語言選擇)

而加喺你既 Desktop 搵 Install Parrot，
再將佢打開，
打開後佢會叫你揀 Languages，
我會用 英文 (美國)，
然後再 Next。

安裝 (國家選擇)

因為你上面係揀左美國，
咁下一個選擇佢就幫你自動揀左國家係美國，
我地轉返佢做你想要既國家。

安裝 (鍵盤設置)

呢度可以唔洗郁，
因為我地平時用緊既 Keyboard 就係 105-key，
或者如果你唔係既話可以再自己轉，
但係我唔轉喇，
撳 Next。

安裝 (選擇系統安裝位置)

喺安裝既過程，
實體機同虛擬機既最大分別就係呢個位，
因為你實體機通常都係要雙系統先要郁呢個位 (係最重要既位置)，
而你既虛擬機係唔需要搞咁多野，
因為呢個虛擬硬碟得一個，
冇其他野 (除非你改左野)。

我地撳 Erase Disk
將所有野刪除再安裝。

然後呢度會有個 Encrypt system
可做可唔做，
呢度 Tick 左之後會要求你打密碼，
然後每次你都要打岩密碼先至入到個系統，
但係我唔需要呢啲，
所以我 Next了。

安裝 (用戶設置)

呢度打返你想起既密碼，用戶名就可以下一步。

安裝 (確定你輸入的內容)

呢度係確定返你上面啲野，
確保係冇錯就可以撳 Install

安裝 (二次確定)

撳 Install now

真係裝喇

而加只需要等佢自己裝好哂。

安裝完成

見到呢個畫面就係代表已經裝好，
撳 Done 重新開機。

移除存放裝置 iso

呢個位我地要將原本既 iso 移除左佢，
以免之後佢入左你隻 iso 先，
唔咁做就會一直都會係安裝畫面。

步驟:
對住你既 vm 右鍵 > 設定 > 存放裝置 > 控制制 IDE > 對住個 ISO 右鍵 > 移除附件 > 移除。

開機測試

當你完成左移除 iso 後，
我地再試下開機睇下成唔成功。

當你見到呢個畫面既時候，
就成功了。

登入

入返你岩岩打左既密碼。

成功安裝

得左 A_A
