VMware - 安裝虛擬主機軟件 (Windows)

發表於2022-06-21|更新於2022-06-21|VMware

VMware

VMware Workstation 於 1999 年推出，是 VMware 推出的第一款產品。
該軟件套件允許用戶在單個物理個人計算機上運行 x86 或 x86-64 兼容操作系統的多個實例，並且每個系統內的都會有一個獨立的隔離，不影響實體機。

【程式名稱】：VMware Workstation
【下載網站】：按這裡
【腳本語言】：多國語言

下載 VMware Workstation 安裝檔

按下上方所說明的 下載網站 內下載檔案以 系統管理員身份 將他開啟。

開啟檔案

VMware Setup Wizard

按 Next。

License Agreement

將 i accept the terms in the License Agreement Tick，然後按下 Next。

Custom Setup

路徑如果不改變就直接按下 Next 進入下一步操作。

Settings

  • Check for product updates on startup // 意思是每當電腦啟動的時候會檢查軟件更新。
  • Join the VMware Customer Experience Improvement Program // 意思是加入 VMware 客戶體驗改善計劃
我不想電腦每次啟動也加載程式，這裡會令電腦慢慢變慢。
我不想加入 客戶體驗改善計劃 這樣會沒有私隱。
所以我兩者也會取消 Tick。

我兩種也不需要，所以將兩種去除 Tick，然後下一步 Next。

Shortcuts

我只需要 Desktop 的桌面連結。

準備安裝

確定以上的設置後，按下 Install 開始真正安裝 VMware。

安裝中 …

安裝完成

安裝完成，按下 Finish。

啟動 VMware Player

直接在右下角尋找 vmware。

License 輸入？

由於我沒有購買，只是免費使用，所以直接按 Continue。

確定完成

啟動成功
