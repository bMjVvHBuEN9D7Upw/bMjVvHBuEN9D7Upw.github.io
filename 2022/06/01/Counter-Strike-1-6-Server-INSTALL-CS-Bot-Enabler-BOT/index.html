Counter-Strike 1.6 - 安裝電腦對打 (CS Bot Enabler)

發表於2022-06-01|更新於2022-06-13|Counter-Strike 1.6

CS Bot Enabler

為你的伺服器新增電腦，

特徵

  • Linux Support
  • 電腦更聰明
  • 任何地圖和模式都支援

Github

  • https://github.com/Arkshine/CSBotEnabler

下載

  • https://github.com/Arkshine/CSBotEnabler/releases/download/v1.0.1/csbot_enabler_mm-1.0.1.zip (主要檔案)
  • https://github.com/Arkshine/CSBotEnabler/blob/master/resources/bot_profiles.zip?raw=true (聱音檔)

安裝

先將剛剛下載的兩個檔案解壓

位置請跟以下

複製至 cstrike 內

將四個檔案,資料夾放在 cstrike 內。

  • addoons (資料夾)
  • sounds (資料夾)
  • BotChatter.db (檔案)
  • BotProfile.db (檔案)

於 Plugins.ini 新增一行

到 \steamapps\common\Half-Life\cstrike\addons\metamod\plugins.ini 內，
新增一行 (Windows or Linux)，

Windows

Linux

然後儲存。

完成

完成以上的操作後，
請開啟你的伺服器，
然後在控制台打。

注意 :
每當一個新地圖加入電腦的時候，
都需要小小時間載入 Bot 的數據，
載入的時候有可能會彈出伺服器，
但不用擔心，
只需要再次進入伺服器即可。

另外，每次輸入一次 bot_add = 加入一個電腦。

載入完成後 …

配置

設置電腦的無線電。

預設為 0 = 關閉
1 為開啟

到 csbot_enabler/config.ini

指令

更多指令可以以下兩句查看。
