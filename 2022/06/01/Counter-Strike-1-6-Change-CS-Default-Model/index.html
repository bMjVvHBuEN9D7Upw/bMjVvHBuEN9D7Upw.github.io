Counter-Strike 1.6 - 更改為 CS 模型

發表於2022-06-01|更新於2022-06-07|Counter-Strike 1.6

前言

原本 CS1.6 既 Model 係用緊 CZ (見圖下)，
呢種既 Model 係大家認識既 CSO Model，
但其實佢係由 CZ 出黎既 Model (Counter-Strike: Condition Zero)，
呢隻 Model 我並唔鍾意用，
所以今日會講點樣改返做 CS1.6 既 Model。

開始

我地需要開啟 CS1.6 既 Optinos，
然後再將撳 Video，
再將原本 Tick 左既 Enable HD Models if available，
Disable 左佢。

重開後就會見到唔見左 CE Model。
