如果你想加入自訂既圖，你可以去呢個網度搵，

https://fontawesome.com/search

說明二 : || 係必須要隔開，然後後面打你想要既 Fontawesome code。

子分類新增說明

如果你想加其他野喺 Navbar 入面，例如係 :

  • 你既分類入面既 link
  • 你既tags 入面既 link

咁樣仲簡單，只需要入返你既 tags / categories link，
再將入完既 link 擺入去頭先 menu: 既位置，再重開 server 就加到架啦。

以上操作唔需要 add page。

  • Counter Strike 1.6
  • Counter-Strike Global Offensive

然後想更新子分類見到既 display 畫面，就將 server 重開。
