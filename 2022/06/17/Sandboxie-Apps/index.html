Sandboxie - 虛擬沙盒隔離軟件

發表於2022-06-17|更新於2022-06-23|Windows

說明

Sandboxie 是一款沙盒的隔離軟件，適用於 32 位和 64 位基於 Windows NT 的操作系統。
可以創建一個類似沙盒的隔離操作環境，可以在其中運行或安裝應用程序，而無需永久修改本地和映射驅動器或 Windows 註冊表。
隔離的虛擬環境允許對不受信任的程序和網上沖浪進行受控測試。

官方網站

Github : https://github.com/sandboxie-plus/Sandboxie
網站 : https://sandboxie-plus.com/

版本

Sandboxie 有分開兩個版本，另外如果安裝 Plus 版本是會幫你安裝 Classic 版本的，兩者都可以使用。

  • Sandboxie Plus
  • Sandboxie Classic

特徵

兩個非常強又實用的功能 :

  • 防火牆 - 每個沙盒都有一個新的防火牆
  • 快照管理器 - 可以即時還原工作程序
