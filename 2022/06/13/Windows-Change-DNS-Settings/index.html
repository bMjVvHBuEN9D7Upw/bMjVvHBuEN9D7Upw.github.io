Windows - 改變 DNS 位置

發表於2022-06-13|更新於2022-06-13|Windows

什麼是 DNS？

根據 Cloudeflare 所說 :

網域名稱系統 （DNS） 是網際網路的電話簿，
人們透過例如 nytimes.com 或 espn.com 等網域名稱線上存取資訊。
Web 瀏覽器透過 網際網路通訊協定 （IP） 位址進行互動。
DNS 將網域名稱轉換為 IP 位址，以便瀏覽器能夠載入網際網路資源。

為何要改變 DNS 位置？

原因一 : 私隱問題

我們預設使用的 DNS 位置是由網路供應商提供的，
這個意味著如果這個位置是有用到記錄的操作，
網路供應商是可以知道你在上什麼網站。
因此，我們要使用其他 DNS 位置，
將你上網的匿名性加強。

原因二 : 速度問題

加強你上網的速度，
使用 Cloudflare DNS 使你上網速度更快，
公司聲稱會於24小時內由系統自動刪除記錄，
然而世界上最快的 DNS 位置是由 Cloudflare 提供的
1.1.1.1 和 1.0.0.1 位置 (ipv4)。

原因三 : 保安問題
使用不同的 DNS 位置，
例如 VPN 供應商提供的 DNS 位置，
他們會於伺服器內設立防火牆，
當你可能進入了他們的黑名單網站 (有毒網站)，
這樣就可以防止你進入了不明網站 (有毒網站)，
從而加強了你上網的保護。

為何不使用預設 DNS 位置？

簡單一句說完，
就是以上的三個原因，沒有一樣是預設 DNS 有的。

  • 他會追蹤你上網
  • 可以隨便封鎖你上什麼網站
  • 速度不及其他 DNS Server
  • 沒有為你設立任何的保安

開始教學 (前言)

本篇文章使用 Windows 10 系統示範。

以下教學如果你是使用 Windows 7 / 11 系統的話，
方法大同小異，
分別是 網路和網際網路 的位置不同的開啟方法，

開啟設定

首先請按右下角電腦的
然後再按

網路和網際網路

到此畫面後，請按下 網路和網際網路。

變更介面卡選項

到此畫面後，請按下 變更介面卡選項。

網路連線

到此畫面後，請尋找你上網的位置 (通常是 乙太網路)，
找到之後請跟以下步驟 :

注意 : 以下步驟需要使用 系統管理員

  1. 按下右鍵
  2. 按下最底的 內容(R)

乙太網路 內容 (1) - IPV4

到此畫面後，請在中間拉下去尋找這個東西 :

  • 網路網際通訊協定第 4 版 (TCP/IPV4)

乙太網路 內容 (2) - IPV4

找到之後請對他左鍵一下，
然後再按下面的 內容(R)

網路網路通訊協定第 4 版 (TCP/IPV4) - 內容

到此畫面後，是真正修改 DNS 位置的時候了 ~~

網路網路通訊協定第 4 版 (TCP/IPV4) - 修改

按下 使用下列的 DNS 伺服器位置(E):
接下來你可以選擇你想用的 DNS 供應商。

Google Public DNS IP addresses

Google 公共 DNS IP 地址 (IPv4) 如下：

  • 8.8.8.8
  • 8.8.4.4

輸入完成後按下確定即可。

Cloudflare 公共 DNS IP 地址 (IPv4) 如下：

  • 1.1.1.1
  • 1.0.0.1

輸入完成後按下確定即可。

乙太網路 內容 (1) - IPV6

接下來我們跟剛剛一樣做法，
這次是修改 IPV6 位置，
將剛剛的視窗關閉，
然後尋找 :

  • 網路網際通訊協定第 6 版 (TCP/IPV6)

乙太網路 內容 (2) - IPV6

按下 內容(R) 後，
按下 使用下列的 DNS 伺服器位置(E):

Cloudflare 公共 DNS IP 地址 (IPv6) 如下：

  • 2606:4700:4700::1111
  • 2606:4700:4700::1001

輸入完成後按下確定即可。

Google 公共 DNS IP 地址 (IPv6) 如下：

  • 2001:4860:4860::8888
  • 2001:4860:4860::8844

輸入完成後按下確定即可。

完畢

教學完畢。

以下整合位置 :

Cloudflare 公共 DNS IP 地址 (IPv4) 如下：

  • 1.1.1.1
  • 1.0.0.1

Cloudflare 公共 DNS IP 地址 (IPv6) 如下：

  • 2606:4700:4700::1111
  • 2606:4700:4700::1001

Google 公共 DNS IP 地址 (IPv6) 如下：

  • 2001:4860:4860::8888
  • 2001:4860:4860::8844

Google 公共 DNS IP 地址 (IPv4) 如下：

  • 8.8.8.8
  • 8.8.4.4
