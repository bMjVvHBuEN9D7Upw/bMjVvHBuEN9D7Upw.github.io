Windows 系統 - 移除 Discord 開機啟動程式

發表於2022-06-13|更新於2022-06-13|Windows

前言

每當電腦開啟的時候，
Discord 也跟著會一起開的時候，
這個過程會令你的電腦減慢了，
所以今篇文係教你點樣將佢較到唔好開機啟動。

方法一 (簡單)

使用 Discord 設置提供既方法，
呢個方法好簡單，
但係佢會繼續存在喺你部機度。

開始 (Settings)

登入你既 Discord，
撳下面既

Windows 設定

黎到呢個位置喺左邊搵 Windows 設定。

系統啟動行為

將開啟 Discord 熄左佢，咁就唔會再開機啟動，
當然你想開返只要開返佢就得。

方法二

從你的 Windows 系統內將 Startup 檔刪除，
呢個方法比較複雜，
但係佢唔會再出現喺你部機度。

Win + R ()

我地喺你既鍵盤度撳 Win + R

然後輸入

刪除 Startup 檔案

之後佢會幫你開左個 Explorer 然後入左呢個 Paths。

我地搵 Discord Inc，
然後將佢成個 Delete，咁就成功唔會再開機啟動了。
