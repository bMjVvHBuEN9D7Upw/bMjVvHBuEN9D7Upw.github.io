Discord - 關閉開機自動開啟程式

發表於2022-06-13|更新於2022-07-06|Discord

Discord 預設

當你安裝左 Discord 之後，其實 Discord 本身既設定係會幫你整埋每當你開機既時候都會自動幫你開 Discord，咁樣既設定固然係方便，但係當你部機係老爺機，又或者程式越黎越多既時候，呢個自動開啟會變成你電腦既負擔，繼而電腦變慢。

Windows Regedit 說明

Discord 本身都有提供一個方法俾你唔好再開機就開 Discord，其實佢個原理只係幫你喺 regedit 入面 delete 一樣野，而路徑係：

電腦\HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run

上面呢個路徑入面既內容就係你呢部機開機之後會跟住開既程式。

說明：只要將佢 delete 就會失效唔再開機會啟用，而新增就會幫你開機啟動程式。

  1. 首先登入你既 Discord
  2. 撳下面既

Windows 設定

  1. 黎到呢個位置喺左邊搵 Windows 設定。

系統啟動行為

  1. 將開啟 Discord 熄左佢，咁就唔會再開機啟動，當然你想開返只要開返佢就得。
