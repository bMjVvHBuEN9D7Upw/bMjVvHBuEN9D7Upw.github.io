Cyber Security - 安裝作業系統後必須要做的安全事項

發表於2022-06-11|更新於2022-06-11|三修系統大撚哂

前言

當你剛剛安裝好你的作業系統後，
以安全角度是並未做好設置，
你必須進行以下操作來為你的系統有更進一步的安全。

第一樣

  • 本機帳號加入 一般用戶 權限的帳號

安裝後，
部份系統是沒有為你建立普通權限的本機帳號，
像是 Windows 這樣的作業系統，
預設是使用 系統官理員 的帳號，
他並沒有為你建立普通帳號，
當你什麼事情也是使用 系統官理員 身份進行，
在電腦世界是相當危險，
所以你要做的是，
為你的系統建立一個 一般權限 的帳號，
平時使用 一般權限 的帳號，
需要使用 官理員 權限才切換帳號，
這樣才是最好的做法。

第二樣

  • 瀏覽器安裝插件

你平時上網的時候，
每個網站也有各種不同的插件和腳本，
你並不知道當中加入了什麼，
有可能是有害你電腦的插件，
有毒的廣告，木馬等等。

建議安裝以下插件。

Privay Badger

將追蹤網站封鎖。

ublock Origin

封鎖不同的有害連結。

HTTPS Everywhere

將你瀏覽的網站加入 HTTPS，
將你和網站的連結加多一層保護 (加密連線)，
防止 Mitm Attack (中間人功擊) 等。

以上就是我個人認為必須要做的事情。
