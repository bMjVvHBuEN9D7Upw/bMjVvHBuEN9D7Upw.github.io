XAMPP - 安裝 Apache 於 電腦上 (Windows)

發表於2022-06-22|更新於2022-06-23|XAMPP

什麼是 XAMPP？

XAMPP 是 Apache Friends 開發的免費開源跨平台 Web 服務器解決方案堆棧包，主要由 Apache HTTP Server、MariaDB 數據庫和 PHP 和 Perl 編程語言編寫的腳本解釋器組成

【程式名稱】：XAMPP
【官方網站】：按這裡
【下載檔案】：按這裡
【程式用途】：於本機位置建立 Apache HTTP Server (localhost)

下載 XAMPP 安裝檔案

進入官方網站後，尋找最新版本下載檔案，見圖下。

開啟 XAMPP 安裝檔案

開啟後，他會要求你用 系統管理員身份 開啟。

會提示你的路徑擺位，按下 OK 後繼續。

Setup XAMPP

來到 setup Wizard 後，按下 Next。

選擇安裝的 Components

這邊直接全都要就好，以免後來你需要使用的時候又要再安裝多一次，所以直接按下 Next 後繼續。

選擇安裝路徑

這邊直接使用預設的 C:\xampp 就好了，如果用其他有可能會有權限問題令到你不能開啟 localhost，所以是一樣，直接按下 Next。

選擇語言

XAMPP 是沒有繁體中文的，所以直接使用英文吧，然後按 Next。

Bitnami for XAMPP

這邊如果你會使用 Wordpess 這些的話是有用的，但如果沒有就直接跳過吧，加上如果要安裝我本人也建議直接下載 wordpress 的檔案在連結中安裝 wordpress，將 checkbox uncheck 後，按下 Next。

準備安裝

確定以上的設定後，開始真正的安裝，按下 Next。

安裝中 …

靜心等待他安裝完成。

防火牆設定

這邊去到最後的時候，會彈出一個視窗要求防火牆權限，將 私人網路 Tick 後，按下 允許存取(A) 後繼續。

安裝完成

已經成功在你的電腦中安裝 XAMPP。

XAMPP Control Panel

這邊我不建議使用系統管理員身份開啟 XAMPP，所以我會直接吧程式關掉再開。

注意 : XAMPP 視窗關掉後不是真正關掉了，你要在右下角的 ^ 找 XAMPP 再對住他右鍵將他關閉才真是正關閉 XAMPP。

開啟 Server

直接在 XAMPP Control Panel 的 Apache 按下 Start。

進入網頁

直接在瀏覽器輸入 localhost。
