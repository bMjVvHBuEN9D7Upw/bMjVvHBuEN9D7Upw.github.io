發表於2021-12-13|更新於2022-07-14|Metasploit

需要工具

如要成功令目標電腦受操控於你，必須開放目前主機的 IP 位置並且開放一個 Port 位置讓其他人可以進入到你的伺服器。
本篇文章會需要使用以下的工具：

  • VPS Server - 本篇文章會使用 Debain VPS Server 主機。
  • Metasploit - 生成木馬，操控電腦
  • Screen - 後台運行伺服器程式，避免退出程式後會關閉操作。(重要)
  • ufw - 設置 Open Port (必要)
  • curl - 檢查 Public IP 位置 (可選)
  • 對使用 Linux 指令有一定純粹的程度的頭腦 - 沒有的話接下來你會跟得很辛苦，可說是跳級挑戰。
  • Apache - 讓人進入你的網址再下載你的後門程式 (可選)

最後一步 - 等待某人開啟後門程式

確定後你就可以輸入以下指令開始監聽，等待某人開啟你的後門程式，然後當有人開啟檔案你這邊的控制台就會有反應，然後就成功入侵電腦。

要進行監聽方法有幾種，第一種是不能在同一個視窗中進行工作，即是輸入指令，你只能等待到有反應才能繼續使用指令。
第二種是你可以繼續於同一個視窗中繼續工作，即是輸入指令，而且一樣會是即時有反應，不同的是你需要手動進入到該 Session。

Exploit 方法一 - exploit

等待的時候，如果你真的不想再等待，你可以直接按下鍵盤 Ctrl+C 停止。

左邊是入侵者畫面，右邊是受害者畫面，當受害者電腦開啟檔案後，入侵者的畫面就會有反應，此事此刻開啟你後門程式的電腦已經完全受入侵者控制。

Exploit 方法二 - run

等待的時候，如果你真的不想再等待，你可以直接按下鍵盤 Ctrl+C 停止。

左邊是入侵者畫面，右邊是受害者畫面，當受害者電腦開啟檔案後，入侵者的畫面就會有反應，此事此刻開啟你後門程式的電腦已經完全受入侵者控制。

Exploit 方法三 - exploit -j -z

同步進行，使用這指令你可以繼續在指令台中工作，但問題是你需要手動進入至該 session。
