Metasploit - 外網木馬生成 (Windows exe)

發表於2021-12-13|更新於2022-06-14|Metasploit

前言

以下教學全程會用指令操作所有野，
學好指令點打會更加得心應手，
唔係既話，
係會跟得好辛苦。

需要工具

  • Metasploit (Trojan 生成，操控中毒者電腦)
  • Linux 系統 (本文章使用 Ubuntu)
  • ufw (Linux 防火牆)
  • curl (下載 Metasploit,查你既 Public IP)

curl 安裝

首先我地要查下你既 Public IP 係乜，
打 curl 睇下有冇裝到，
部機如果冇，
即係冇裝到，
咁打返呢句裝返 ~~

查 Public IP

安裝 ufw

ufw 係一個防火牆工具，
我地會用佢黎 set open port (tcp)。

啟用 ufw

當你成功安裝 ufw，
預設係唔會為你開啟 ufw，
我地需要用指令去為佢開啟返個程式。

開啟 Port 位置

今次我會用 tcp 同埋 7777 Port Number.

Wan Trojan 生成

用 msfvenom 生成 後門 exe

  • LHOST 你既 Public IP Address
  • LPORT 你岩岩開左既 Port Number
  • fuckyou.exe 是但改個你鍾意既 exe file 名

完成兩個步驟，包括係 :

  • setup open port
  • 生成後門 exe 檔

設置監聽

當成功設置上述既野既時候，
接下來我地要做既係 setup 你既 metasploit 既控制台 (控制人地電腦)。

開啟 msfconsole

開 Terminal 打

Payload 進入

喺呢度輸入兩句。

Payload 設置

而加我地要將自己既 ip 打入去 options 度，
我地而加係要做 Host，
但係 msfconsole 仲未知道你係 Host，
我地喺控制台度打 show options。

查 Private IP

黎到呢個位我地開另一個 Terminal 查我地而加既 Private IP 係乜，
再將查到既呢個 Private IP 輸入去 msfconsole。

安裝 net-tools

打 ifconfig 查我地既 lan ip address係乜

如果你部機打 ifconfig 冇反應，
咁打返呢句裝返。

ifconfig

我地喺 Terminal 打以下指令，
搵第一個既 inet，就係我地既 LAN ip addrss，
查到之後就將你既 LAN ip address 打入去。

輸入 Private IP

改埋你岩岩開左既 Port Number

確定數值

做一個確定。
我地睇下個 set 既野岩唔岩 ~

開始監聽

確定哂之後你就可以輸入以下指令開始監聽，
等待某人開啟呢個程式，
然後呢邊 (Server)，就會有即時反應，
然後就成功入侵 ~

入侵成功

見到佢開左就會自動有反應

去到呢個位你就可以搞破壞啦 ~~
