發表於2021-12-13|更新於2022-06-30|Metasploit

前言

以下教學全程會用指令操作所有野，學好指令點打會更加得心應手，唔係既話，係會跟得好辛苦。

需要工具

  • Metasploit (Trojan 生成，操控中毒者電腦)
  • Linux 系統 (本文章使用 Ubuntu)
  • ufw (Linux 防火牆)
  • curl (下載 Metasploit,查你既 Public IP)

curl 安裝

首先我地要查下你既 Public IP 係乜，打 curl 睇下有冇裝到，部機如果冇，即係冇裝到，咁打返呢句裝返 ~~

查 Public IP

Print 既 result 就係你既 Public IP Address。

安裝 ufw

ufw 係一個防火牆工具，我地會用佢黎 set open port (tcp)。

啟用 ufw

當你成功安裝 ufw，預設係唔會為你開啟 ufw，我地需要用指令去為佢開啟返個程式。

開啟 Port 位置

今次我會用 tcp 同埋 7777 Port Number.

Wan Trojan 生成

用 msfvenom 生成 後門 exe 檔案。

  • LHOST 你既 Public IP Address
  • LPORT 你岩岩開左既 Port Number
  • fuckyou.exe 是但改個你鍾意既 exe file 名

完成兩個步驟，包括係 :

  • setup open port.
  • 生成後門 exe 檔案。

查 Private IP

黎到呢個位我地開另一個 Terminal 查我地而加既 Private IP 係乜，再將查到既呢個 Private IP 輸入去 msfconsole。

安裝 net-tools
