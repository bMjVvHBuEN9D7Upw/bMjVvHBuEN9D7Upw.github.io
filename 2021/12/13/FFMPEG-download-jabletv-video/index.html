FFMPEG - 下載 Jable.tv 影片教學 (直至 2022.6.30 仍然可用)

發表於2021-12-13|更新於2022-06-30|FFMPEG

2022.6.16 訊息

我發現原來 youtube-dl 同 yt-dlp 係有 support Jable.tv 既片，
所以以下咁多操作係唔洗做都得 =.=
但係以下既下載方法都係仲 work，當學一個操作囉，反正呢個方法有好多網都做到，唔一定係 Jable.tv 先可以 .w.

前言

超級詳細教學下載 Jable.tv 片教學，方法大同小異。

需要既工具有以下：

  • FFMPEG (下面我會放 Download Link)
  • 你既 Terminal (用黎打指令)
  • 一個正常 Browser

FFMPEG 下載

FFMPEG Download Link : 撳呢度

注意：有機會失效，我只係 Copy 條 Download Link 俾你一撳就 Download，改左自己再上 FFMPEG 搵。

安裝 FFMPEG

  1. Download 完用你既解壓程式去開 (eg: 7zip)。
  2. 我地淨係需要用入面既其中一個檔案，入 ffmpeg-4.4-full_build > bin，你會見到 ffmpeg.exe，拉呢舊檔案出黎用。
    今次既教學係唔需要用到其他個兩舊，然後呢一步你可以選擇做定唔做，建議跟住做 (因為方便你日後打指令)。

設置 Paths

點解要設置 Paths，係因為如果你唔設置既話，你就要慢慢用 cmd cd 入去你個 FFMPEG 既路徑，咁樣做會哂你好多時間。

開啟 sysdm.cpl (環境變數)

  1. 首先，電腦 Search sysdm.cpl，撳完會需要用到系統管理員，你打返密碼入去就 OK。

  2. 再撳 環境變數(N)

  3. 然後我地要加野喺 系統變數，喺入面拉落d搵一個叫 Path 既野，搵到之後撳一下再編輯。

新增變數內容

  1. 將你岩岩拉完既檔案 ffmpeg.exe，放去你想放既位置。
  2. 例如你想放喺 D:\Program Files\Tools，咁你放完係呢個位入面。
  3. 再去返岩岩編輯環境變數個位撳 新增。
  4. 再新增返你 Folder 既 Path，即係打 D:\Program Files\Tools。

尋找 m3u8 連結

  1. 開你既 Browser (我用 Firefox)。
  2. 撳你鍵盤既 F12 (Inspect)。
  3. 之後撳 Network。
  4. 再上 Jable.tv。
  5. 搵你想下載既片，呢度我用既例子係 https://jable.tv/videos/ipx-258-c/。
  6. 喺 Filter URLs 打 m3u8 極速搵條 link，當你成功搵到 m3u8 link 既時候，Copy 條 link：
    https://qto-sara.mushroomtrack.com/hls/lZV1tlFoDmHKmafcXbv3BA/1656584219/0/108/108.m3u8，呢條就係我地要既Link。
注意：搵唔到 m3u8 link 可能係因為你做漏左步驟，撳 F5 Refresh 再試下 Play 條片就一定會搵到架啦。

下載影片

而加我地需要用返岩岩我地裝完既 ffmpeg.exe，然後需要使用指令操作全程，
選擇你想要用既 Terminal (呢度我會用 Windows Terminal 做示範)。

選擇下載位置

然後呢度就選擇你要 Download 既位置，
假設原本我地要去 由 C: 去 D:/Video，就先打 D: 再打 cd Video。

確定位置後下載影片

黎到呢個位我地開始要真係下載條片，將下面條 code 放落你既 Terminal 然後 enter，咁就真係開始 download 緊條片。
另外，順便講解下條 code，我地會用一樣野黎成立我地既讀取影片再下載：
-user_agent // 製造一個假 User-Agent
-c copy // 幫個 file 改個名

注意：如果你唔打呢句，因為 Jable.tv 本身係有處理過條片，普通既下載方法係唔得，我地加埋 User-Agent 喺入面就會解決到 403 既問題。
當你見到 [https @ 000001947ece8180] Opening 'https://qmm-truts.mushroomtrack.com/hls/NFb_sIwTdg3IZ6FeEFo9-g/1656586701/0/108/1082.ts' for reading 呢啲野就代表已經讀取成功，開始下載緊。

注意

呢個等待既時候，你會見到多左個 files 叫 IPX-258.mp4，千奇唔好以為已經 downlaod 完，其實佢仲 download 緊，唔好中途冇拉拉閂左個 Windows (CMD)，咁會炒車搞到你要由頭再 down 過，又或者可能會喺 Download 到個個 time 既位完左條片。
當你等完之後，咁就搞掂哂啦，慢慢打你既 J 啦 hehe ~~

提示：如果你想整做 part file，等下次再繼續 download 條片既話，好可惜 FFMPEG 就好似冇呢個功能 ._.
