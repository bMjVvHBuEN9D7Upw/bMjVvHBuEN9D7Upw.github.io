發表於2021-12-16|更新於2022-01-31|Windows

Rufus

簡介

Rufus 係一個幫你格式化同埋Create一個可以啟動 USB 驅動器既 Program.

官方網站 : Rufus
Github : Rufus Github

Yumi

官方網站 : Yumi
Github (Unofficial) : Yumi Github

簡介

Yumi (Your Universal Multiboot Integrator)
係一個可以Create多個系統 USB 驅動器既Program，
開機既時候會顯示選單，有 Windows, Linux, 分割複製硬碟, 查殺病毒等既系統

程式畫面

注意

YUMI 有分兩個版本 一個係 Legacy Support 一個係 UEFI Support，
要睇返你部機係行到邊個再決定你要下載邊個exe Create Bootable

  • YUMI Legacy Suppor NTFS 或 Fat32 格式，只可以 BIOS USB 啟動
  • YUMI UEFI 只會 Support Fat32， BIOS 同 UEFI USB 啟動（取決於發行版）

版本分別

YUMI Legacy (BIOS USB Boot Only)

如果你部機 Support BIOS 動動，
並唔諗住喺 UEFI 模式下從 YUMI 運行 Windows 安裝程序，
就可以用呢個版本，儘管 CMS Legacy 模式，
大多數現代老母版都仲係支持 Legacy BIOS 固件

YUMI UEFI (BIOS and UEFI USB Boot)

YUMI UEFI 變體係利用 GRUB2 進行 UEFI 和 BIOS 引導，
注意，呢個 UEFI 版本同舊版本並唔兼容
仲有，你既 USB 驅動器必須係 Fat32 格式先會 Support 喺 UEFI 模式下啟動
佢會利用 fat32format 實用程序將大於 32GB 的驅動器格式化為 Fat32

官網傳送門 : https://www.pendrivelinux.com/yumi-multiboot-usb-creator/
