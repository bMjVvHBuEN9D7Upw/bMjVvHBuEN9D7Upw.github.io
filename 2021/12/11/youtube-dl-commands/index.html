Youtube-dl - 下載影片或音樂

發表於2021-12-11|更新於2022-06-30|youtube-dl

Youtube-dl

Youtube-dl 係一個可以 download 好多唔同網站既音樂, 片或者字幕既一個免費開源工具。

第三方程式

因應 youtube-dl 係開源 github 作品，所以出現左好多唔同版本由社群創造既版本，以下係三個作品既分別資訊

版本 更新速度 連接方法 支援系統 指令分別 介面
Youtube-dl 超慢 單線程 Linux, Mac, Windows 唔夠 yt-dlp 多指令 指令操作，白色字顯示
yt-dlp 頻繁 多線程 Linux, Mac, Windows 比起 youtube-dl 多指令 指令操作，顏色字顯示
youtube-dl-gui 似乎已停更 Unknown Mac, Windows 圖形用戶介面 圖形用戶介面

安裝 Youtube-dl

以下會說明使用 curl 或者 wget 的下載方法，轉自作者。

注意 : 如果要用 youtube-dl 轉檔功能，你需要裝 FFmpeg。
安裝方法只需將 FFMPEG 放在和 youtube-dl 同一個資料夾內就能偵查到 FFMPEG。

此方法只適用於你部機本身有裝 curl 工具。

使用 wget 下載

Linux Package Manager 下載

適用於 : MacOS
適用於 : Arch Linux,Manjaro,EndeavourOs
適用於 : Fedora

Windows 下載

你必須已經安裝 Microsoft Visual C++ 2010 Service Pack 1 Redistributable Package (x86)。

https://ytdl-org.github.io/youtube-dl/download.html

更新 youtube-dl

當某網站更新後，有可能會下載失敗，此時可以嘗試將 youtube-dl 更新後再重複輸入指令嘗試下載。

使用以下指令限制使用連線 ipv4。

通常會用呢個指令係因為你下載緊個陣發現出左 403 Error。如果你輸入呢句指令就有機會解決問題，繼續下載你想下載既野。
