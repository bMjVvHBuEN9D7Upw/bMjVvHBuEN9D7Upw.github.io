Aircrack-NG - 爆鳩隔離屋 wifi 密碼 XD

發表於2021-12-11|更新於2022-07-03|Kali

Aircrack-ng

Aircrack-ng 是一套完整的評估 WiFi 網絡安全的工具。

它專注於 WiFi 安全的不同領域：

  • 監控：數據包捕獲和數據導出到文本文件以供第三方工具進一步處理。
  • 攻擊：通過數據包注入重放攻擊、解除身份驗證、虛假接入點等。
  • 測試：檢查 WiFi 卡和驅動程序功能（捕獲和注入）。
  • 破解：WEP 和 WPA PSK（WPA 1 和 2）。

所有工具都是命令行，允許編寫大量腳本，許多 GUI 都利用了此功能。
主要適用於 Linux，也適用於 Windows、macOS、FreeBSD、OpenBSD、NetBSD，以及 Solaris 甚至 eComStation 2。

【操作方法】：使用終端機全指令操作
【作者 Github】：https://github.com/aircrack-ng/aircrack-ng
【文章使用系統】：Kali Linux

需要硬件

如果你要用 Aircrack-NG 既任何一款工具，你必須要有 Wireless Card / Wireless USB。

如果你本身部機用 Lan 線黎上網，無任何 Wireless Card 既話，咁以下既爆破行為你都唔會做到，建議買隻 USB。
你有 Wireless Card 既話，喺 Ethical hacking 層面黎講係好多野都可以做到。相反，冇既話好多野都會限制左你。

檢查網卡狀態

如果顯示唔到好似下圖咁既文字，咁就代表失敗，失敗就代表做唔到以下既操作 :(

說明：通常都係會 display wlan0，唔係既係下一步轉一守個名就得。
  • ESSID : WIFI 名稱，但由於我未入，所以顯示 off/any。
  • Access Point : 你去既 Wifi 既 MAC Address，但由於我未入 wifi，所以會顯示 Not-Associated。
  • Mode : 我地而加張卡係處於一個咩狀態 (未開係會顯示 Managed)。

啟動 Monitor mode

呢度係咪 wlan0 取決於你打完 iwconfig 出現左既係咩名，有機會唔係 wlan0，咁唔係既話只需要打返佢顯示既名就得。

注意：打完呢句就會入左 Monitor Mode，喺呢個 Mode 底下，會將你 而加張 Wireless 卡 Disconnect。
提示：如果你係用 notebook 進行操作既話，喺冇 lan 線既狀態你係會上唔到網。相反，如果你係用 desktop 進行操作既話，咁你就可以繼續用 lan 線黎上網。

而加我地需要搵你想要既目標，然後我機要拎呢部機既三樣野：

  • ESSID = WIFI名
  • BSSID = WIFI Mac Address
  • CH = 頻度編號

建立 Cap 檔案

  • -d [BSSID]：呢個 Wifi 既 BSSID 係乜呢？
  • -c[number]：呢個 Wifi 既 Channal 數字係乜呢？
  • -w [Cap filename]：你想將呢個 cap file 改做乜野名呢？

功擊裝置

黎到呢一步，我地要攞到呢個 wifi 既 Handshake 或者 PMKID，咁先可以進行爆破同埋斷鳩佢線既程序。

攞既方法只需要將部機進行斷線，然後等佢重新連 wifi，然後就會出現。
而點樣斷鳩佢線？.. 睇埋落去你就會知道點做 A_A

警告：截取既時候唔好將個 windows 閂撚左佢，唔係既話你又要打過係好撚麻煩 =.=

成個 Device 功擊

以下呢個指令會將入面所有機都會踢佢走，踢佢走即係代表冇人會再入到呢個 wifi 上網，而當佢嘗試重新連 wifi 既時候，你就已經成功拎到 handshake :3

警告：如果高調進行功擊，wifi 有異常會造成不必要的影響，請小心喔 ~

格式示範

  • –deauth 0：代表無限功擊。
說明：呢個方法係會喺你停之前都唔會停止功擊，直到你閂 windows 或者鍵盤撳 ctrl + c。
