Ubuntu VPS Server - 建立 Counter Strike 1.6 伺服器並廿四小時開放不停止

發表於2021-12-11|更新於2022-07-10|Counter-Strike 1.6

開啟伺服器需求

如果想自己開 CS 1.6 server 又想俾其他人入到嚟，通常你係要開放自己部機嘅 Public IP (Wan Network) 先得，
但係用自己部機嘅 IP 嚟開 server 實在太唔安全，所以今次我哋會用 VPS hosting 代替，
呢篇文會示範點樣用 DigitalOcean 嘅 VPS 喺 Linux 系統下架設一個 CS 1.6 Server
將會用到以下工具：

  • VPS Server (Linux) - 512 RAM 配置，CS需要嘅配置係非常低，所以用最廢嘅配置都足夠有餘
  • SSH - 連接伺服器工具 (必要)
  • Filezilla - GUI 版本傳輸檔案嘅工具 (上/下載) (可選)
  • SCP - 指令版本傳輸檔案工具 (可選)
  • Screen - 讓伺服器程式可以在後台運行 (必要)
  • metamod - CS 1.6 3rd party mod 嘅原祖，可以用 C++ 寫 metamod plugin
  • amxmodx - 都係 3rd party mod，係用 metamod 寫成嘅，最大分別係可以寫 pawn script 做成 amxx plugin，方便同易過 metamod 好多 (網上大部可以搵到嘅插件都係 amxx 格式)

Q : 止於點樣申請VPS ?
A : 我寫左個 Post 係講點用 Digitalocean 起一部 VPS Server：Digitalocean - 建立 Droplets VPS Server
