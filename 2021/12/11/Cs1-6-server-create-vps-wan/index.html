Counter Strike 1.6 - VPS 建立伺服器 (APT)

發表於2021-12-11|更新於2022-05-20|Counter-Strike 1.6

前言

我地如果要喺cs1.6 咁公開一個IP俾唔同國家既人入黎
我地需要開放自己部機既Public IP 俾人入 (Wan Network)
但係用自己部機黎做Server Host 實在太唔安全
所以今次我地會用 VPS Hosting 做我地既 Server Side.
並且係24小時開放

講完咁多廢話，我地正式開始教學!

Q : 止於點樣申請VPS ?
A : 遲D我再寫一篇教大家點申請 VPS Hosting 既 Post :3

開始~

我地用 SSH 登入我地既 Server (基本上所有Linux 系統 都會本身幫你裝左 ssh)

登入就打呢句

之後第一次入佢會問你 yes 定 no

咁我地緊係打 yes 啦 (唔打yes點入個server呀屌你?)

打 yes

然後我地要將你既 Default Reject Port22 登入既開返做 Accept
呢個動作係為左唔好令你日後入返個 VPS Server 會 Reject左你既登入
