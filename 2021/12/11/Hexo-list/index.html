Hexo - 靜態網站生成器

發表於2021-12-11|更新於2022-06-30|Hexo

關於 Hexo

Hexo 是一個基於 Node.js 製作的一個靜態網站 Framework，他不需使用後台連接數據庫而又能建立一個寫文章的環境。

優點

  • 免後台，減低被入侵風險
  • 簡單快速，使用 Node.js 的 Framework。
  • 非常多外掛提供使用 (hexo 資源多)
  • 一個指令就可以執行全部東西
  • Github Page 支援
  • Markdown 語言支援
  • 多個主題選擇
  • 簡單快速就可以切換主題
  • 使用 npm，因此很多插件提供 (npm 資源多)
  • 容量小，網站速度和性能更快
  • 台灣人製作

缺點

  • 超多支那人使用，這個是非常大的問題 (殘體字)
  • 對於不會指令的人來說不友善，因為你需要使用指令才可以執行操作
  • 需要搭建 NodeJS 環境才可操作下去

安裝需求

如要使用 Hexo 建立 Blog，你必須電腦已經安裝了 NodeJS。

如果不懂安裝 NodeJS 的人可以參考這篇文章：NodeJS - 安裝於 Windows 系統內
