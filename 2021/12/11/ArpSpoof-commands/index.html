ArpSpoof - 一個執行 ARP 欺騙攻擊的工具

發表於2021-12-11|更新於2022-05-20|Kali

ArpSpoof

一種對本地未加密網絡上的其他人執行 ARP 欺騙攻擊的程序。

首要條件要係同一個Network底下先可以發動功擊

要發動功擊我地要搵兩樣野

  • 你而加既 route 既 Gateway IP Address (route 搵)
  • 目標既 Private IP Address (Nmap 搵)

Gateway 尋找

  • 喺屋企既通常都係 192.168.0.1
  • 用VM 既 Default 會係 10.0.0.0 (應該係 唔記得左xd)

要搵 Gateway 既IP 有幾種方法

  • 通常第一行就會係你既 Geteway IP Address

有 d Linux機本身冇 route
咁你打返呢句野裝返就用到

我地可以睇到 Gateway 係寫 192.168.79.2

然後 Iface 係寫 ens33

而加我地要搵埋我地要進行功擊既Device係乜

我地用 nmap Scan 下個 Network 有乜機

部機本身冇Nmap就打呢抽野裝 :

Scan個 Network有邊部機係入左呢個NetWork
