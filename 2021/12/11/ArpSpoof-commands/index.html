ArpSpoof - 一個執行 ARP 欺騙攻擊的工具

發表於2021-12-11|更新於2022-06-30|Kali

ArpSpoof

一種對本地未加密網絡上的其他人執行 ARP 欺騙攻擊的程序。

警告 : 文章的功擊手段，在公眾範圍亂對他人使用，而又被發現是你造成，會有機會造成法律問題，請小心行動。

警告 : 文章的功擊手段，在公眾範圍亂對他人使用，而又被發現是你造成，會有機會造成法律問題，請小心行動。

功擊條件，資訊要求

首要條件要係同一個 Network 底下先可以發動功擊。

要發動功擊我地要搵兩樣野：

  • 你而加既 route 既 Gateway IP Address (route 搵)
  • 你而加既 route 既 Gateway Interface name (route 搵)
  • 目標既 IP Address (Nmap 搵)

Gateway 格式

  • 喺屋企既通常都係 192.168.0.1
  • 公司，學校可以會係 10.0.0.1

但係我地唔會咁傻仔走去慢慢試，所以我會講點樣最直接搵 Gateway 既 IP，
方法有幾種，可以用唔同既指令搵，而今篇我會用 route 去教大家搵 Gateway IP Address。

安裝 net-tools

要查資料既話，首先我地係需要用到接下來講既呢個指令 route，ifconfig，
我地需要確保你部機係已經安裝左以下既套件，我地打以下呢句安裝。

說明 : 即使你覺得你本身已經裝好都唔洗擔心，打多次唔會再裝多個，唔似 Windows 咁。

尋找 Gateway IP Address，Interface Name

喺 Terminal 輸入以下指令。

  • 藍色圈既 192.168.0.1 就係我地要搵既 Gataway IP Addres
  • 紅色圈既 eth0 就係我地要搵既 Inerface name

我地可以睇到 Gateway 係寫 192.168.0.1

然後 Iface 係寫 eth0。

尋找功擊目標裝置 (Scan Network)

說明 : nmap 有好多唔同既參數俾你用，你都可以唔洗跟我咁樣打黎 Scan Network。
  • -sS -O 呢種方法可以列出呢個裝置既 : Ports, Service Name, Mac Addres, IP Address。
  • /24 解整個 Network，如果不輸入 /24 就只有 Scan 指定一個裝置。

我地輸入完呢句指令之後，會發現到好多唔同既裝置，呢度我地要確定兩樣野，包括係你想搞既個部機既 :

  • IP Address (裝置位置)
  • 名稱 (裝置名稱)

確定左係你想打既目標之後就記住佢既 IP ADDRESS，然後去下一個步驟。
