Ubuntu setup 速成 / 倉頡 輸入法 (ibus)

發表於2021-12-20|更新於2022-06-13|Ubuntu

前言

Ubuntu 本身係已經裝好 ibus 速成 倉頡。
我地要做既係需要 enabled 返佢就得，

今次呢篇文會教你點將 ibus 速成改到好似 Windows 咁撳制。

但係點整？
以下落黎就會教你點做 :3

點裝？

開你既 Terminal 打

見到 有 PrintOut zh_HK.utf8 or zh_TW.utf8，
咁即係系統已經裝左中文繁體語言Package，如果冇顯示，
撳 Menu Search Language Support 再安裝返語言套件。

完整影片安裝 Language

裝既時候係需要啲時間，
大概 around 要裝 5 分鐘

當你裝完再 reboot 部機就會生效

Enabled 中文輸入法 (速成 Classic)

留意返 Classic 先係我地 Windows 打開既速成
速成係有分第幾代既 :p

之後點做？

我唔打啦，自己睇片 XDDDD

改做 Windows 設定 (下一頁，殘繁字 List)

因為 ibus 本身既

  • 下一頁
  • 換輸入法
  • 語言係有殘體字
    都唔係好似 Windows 咁係用
    Ctrl + Space 同 Space

預設會係 撳 +- 下一頁上一頁
而預設會係只有繁體字係 list入面

另外，原本我地 Windows 打開既速成入面係有殘體字喺度，
所以我地要做一樣野就係喺 ibus 速成 setting 入面改做繁殘字 list，
咁先係原本 Windows 速成條 List.

我首先會示範改 語言 同埋 撳 Space 換下一頁
睇片 ~~

完成呢步你就可以 撳 Space 換下一頁
字既 List 都會有殘體字，
即係 Windows 既 List ~

改做 Winodws 設定 (換輸入法)

我地原本 Linux 佢 Default 係撳 tab 換輸入法，
但係如果我地想改返做 Ctrl + space 好似 Windows 咁樣，
係需要入去 Settings 改，
右下角，再 Serach Setting。

睇片 :>

完成

當你完成哂以上咁多個步驟就成功改到好似 Windows 咁啦。
但係有樣野係唔得既，
就係加 關聯字，
呢樣野 Linux 咁耐以黎都係未有，
所以大家唔洗好奇點解冇啦 …
我都想有得用 ._.
